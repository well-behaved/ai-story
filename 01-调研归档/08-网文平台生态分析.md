## 黄金三章完整要求

### 第一章 (最关键)

**开头50字规则**:
- 必须有: 冲突/悬念/信息差/异常事件
- 禁止: "我叫XXX，今年XX岁" 式自我介绍
- 禁止: 大段环境描写/世界观说明
- 推荐开局: 危机/穿越时刻/金手指触发/被打脸

**第一章必须完成的事**:
1. 展示主角核心特征(1-2个鲜明特点)
2. 建立主角困境(让读者同情/好奇)
3. 展示或暗示金手指
4. 至少一个小爽点或悬念
5. 字数2000-4000(不宜太长)
6. 章尾强钩子

**禁止事项**:
- ❌ 超过200字的背景介绍
- ❌ 超过3个新角色同时出场
- ❌ 纯对话无冲突
- ❌ 慢节奏日常

### 第二章

**核心任务**:
1. 金手指正式启动/明确展示
2. 第一次使用金手指获得优势
3. 出现第一个小对手/压力源
4. 建立第一个可达成的小目标
5. 展示升级/变强的路径

### 第三章

**核心任务**:
1. 第一次打脸(对看不起主角的人)
2. 获得第一个实质性奖励
3. 建立中期目标(读者知道接下来要干什么)
4. 金手指的第二次使用(确认不是一次性)
5. 尾部钩子引向更大的世界

## 爽点设计方法论

### 8种爽点类型详解

**1. 打脸**:
```
结构: 对方嚣张 → 看不起主角 → 主角出手 → 碾压 → 对方震惊/后悔
变体: 众人面前打脸(最爽) / 暗中打脸(伏笔回收) / 连续打脸(群体)
频率: 每3-5章一次大打脸
```

**2. 升级**:
```
结构: 瓶颈 → 机缘 → 突破 → 实力展示
变体: 战斗中突破(紧张) / 闭关突破(积累) / 顿悟突破(意外)
频率: 每10-20章一次小突破，每30-50章一次大突破
```

**3. 获宝**:
```
结构: 发现线索 → 探索/争夺 → 获得 → 认知价值
变体: 拍卖获宝 / 探险获宝 / 传承获宝 / 意外获宝
频率: 每5-10章一次小收获
```

**4. 装逼**:
```
结构: 低调出场 → 别人小看 → 不经意展示 → 全场震惊
关键: 主角本人不在意(更爽)
频率: 每5-8章一次
```

### 节奏控制黄金公式

```
第一卷(50章为例):
  ch1-3: 密集小爽(每章1-2个) [黄金三章]
  ch4-10: 中等密度(每2章一个中爽)
  ch11-20: 积累+小爽(每3章一个)
  ch21-30: 中期高潮准备+中爽
  ch31-40: 大爽(升级+打脸组合)
  ch41-48: 高潮大战(持续爽)
  ch49-50: 收束+下卷伏笔

爽点密度参考:
  总爽点/总章节 ≈ 0.5-0.8 (每2章至少1个爽点)
```

## 变现模式

### 付费模式 (起点/晋江/飞卢)

```
收入结构:
  订阅分成: 读者付费 × 分成比例(50-70%)
  打赏: 读者打赏 × 分成比例
  全勤奖: 600-1500元/月
  完本奖: 按字数和成绩
  渠道分成: 其他平台销售

起点收入示例:
  均订1000: 约3000-5000元/月
  均订3000: 约10000-15000元/月
  均订10000: 约30000-50000元/月
  大神级(均订30000+): 年入百万+
```

### 免费模式 (番茄/七猫)

```
收入结构:
  广告分成: 按阅读量分成
  计算公式: 千次展示 × 单价(0.5-2元)
  
番茄收入参考:
  新人期: 500-2000元/月
  稳定期(数据好): 3000-10000元/月
  头部作品: 10000-50000元/月
  
特点:
  - 收入上限低于付费模式
  - 但门槛更低，容易起步
  - 适合AI辅助批量生产
```

### 混合模式 (知乎盐选)

```
收入结构:
  稿费买断: 千字50-200元
  或分成: 按阅读量
  
知乎盐选收入:
  单篇5000字: 250-1000元
  单篇10000字: 500-2000元
  高质量故事可被推荐: 额外流量分成
```

## 内容审核规范

### 绝对禁止内容

- 政治敏感（影射领导人/体制批评/分裂言论）
- 色情描写（露骨性描写/未成年相关）
- 暴力极端（过于血腥/虐杀/恐怖主义）
- 宗教冒犯（贬低宗教/邪教宣传）
- 违法犯罪教唆

### 限制内容（可写但需注意）

- 灵异/鬼怪: 需要"科学解释"或"原来是梦"
- 盗墓: 不能美化盗墓行为
- 犯罪: 必须惩恶扬善
- 穿越: 不能写"穿越回真实历史改变结果"
- 同人: 需避免真人同人的法律风险

### 常用变通方法

```
灵异 → "异能" / "精神力感知"
鬼 → "怨灵" / "残留意识" / "信息体"
盗墓 → "考古" / "探险" / "寻宝"
穿越到历史 → "平行世界" / "架空历史"
色情 → fade to black / 隐晦暗示
政治 → 架空国家 / 异世界
```

## AI写网文的现状和机遇

### 当前现状

**成功案例**:
- 番茄/七猫上有稳定AI辅助作者群体
- 日产1-3万字的作者多为AI辅助
- 部分短篇平台(知乎/短篇APP)AI作品过稿率高

**主要问题**:
- 长篇一致性仍是最大瓶颈
- 纯AI文容易被读者识别(重复/无特色)
- 平台检测逐步加强(特别是起点/晋江)
- 角色深度不足，情感共鸣弱

### 机遇分析

```
最适合AI的场景:
  1. 番茄/七猫免费文(质量要求相对低，量大取胜)
  2. 短篇/知乎盐选(单篇完整，无需长期一致性)
  3. 飞卢同人(有原作设定参考，降低创意难度)
  4. 起点辅助写作(人工主导，AI加速)

AI优势:
  - 速度: 日产数万字
  - 一致性(短期): 单章内质量可控
  - 多版本: 同时生成多个方案选择
  - 成本: 比雇枪手便宜
  
AI劣势:
  - 长期一致性差
  - 创意同质化
  - 无法真正理解"爽"
  - 角色缺乏灵魂
  - 被检测风险
```

### 推荐策略

```
保守策略(低风险):
  - AI辅助大纲和结构设计
  - AI生成初稿，人工深度重写(50%+)
  - 使用AI做对话/描写扩展
  - 人工把控关键剧情节点
  
进取策略(中风险):
  - AI生成+人工润色(30%修改)
  - 微调模型降低AI特征
  - 选择宽松平台(番茄/七猫)
  - 避开起点/晋江

激进策略(高风险，不推荐):
  - 纯AI生成，仅人工审核
  - 多号发布分散风险
  - 追求量而非质
```
