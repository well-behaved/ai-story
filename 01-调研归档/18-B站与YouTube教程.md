## 视频教程核心洞察总结

### 1. 变现路径排序(按难度低→高)

```
1. 知乎盐选短篇 (最容易入手，单篇收入200-2000)
2. 番茄/七猫免费文 (日更，广告分成，月入1000-5000)
3. 起点/晋江付费文 (门槛高，但收入高，月入5000+)
4. Amazon KDP英文 (需英文能力，国际市场)
5. 有声书/短剧 (新兴方向，竞争小)
```

### 2. 工具选择共识

```
新手入门: Coze(免费) 或 笔灵AI(中文)
有技术: Python脚本 + API
互动写作: SillyTavern + WorldBook
一致性管理: Dramatica-Flow / Obsidian
批量生产: 非凡写作 或 自建流水线
```

### 3. 避坑提醒

```
视频们共同强调的坑:
  ❌ 以为AI可以完全替代人工
  ❌ 忽视上下文管理(导致崩设定)
  ❌ 不做去AI味处理
  ❌ 在严格平台(起点)发纯AI文
  ❌ 只看播放量不看实际收入数据
  ❌ 忽视读者反馈
```

### 4. 2024-2025趋势

```
从视频趋势可以看出:
  - 2024年初: "AI写小说赚钱"类视频爆发
  - 2024年中: 实战反思类内容增加
  - 2024年末: 工具对比/方法论沉淀
  - 2025年: 开源工具+系统化方法成为主流
  
  从"AI替代人"转向"AI辅助人"的叙事转变
```
