# 商业产品与SaaS工具

AI长篇小说写作相关的所有商业产品和SaaS工具详细记录。

---

## 英文市场

### 1. Sudowrite

- **URL**: https://www.sudowrite.com
- **定价**: $10/月起（Hobby），$25/月（Professional），$100/月（Max）
- **核心功能**:
  - **Story Bible**: 自动维护角色/世界/情节一致性文档
  - **Canvas**: 可视化卡片管理，拖拽组织故事元素
  - **Write**: AI续写，支持多风格选择
  - **Edit**: 包含 Describe / Expand / Rewrite / Feedback 四大编辑工具
  - **Autocomplete**: 分析角色/语调/情节弧线后智能补全
  - **Story Engine**: 从概念到完整小说的自动化流水线
  - **1000+插件**: 社区贡献的写作辅助插件
- **长篇处理方式**:
  - Story Engine将故事压缩为"story DNA"传入上下文
  - Canvas卡片系统管理章节和场景
  - 逐场景生成，维持全局一致性
  - 可生成50k-100k字完整小说
- **用户评价**:
  - 文学质量最高的AI写作工具
  - Story Engine质量参差不齐，需大量引导
  - 适合有经验的作家加速写作
  - 不适合纯自动化

### 2. NovelCrafter

- **URL**: https://novelcrafter.com
- **定价**: $4/月（Hobbyist），$8/月（Storyteller），$20/月（Wordsmith）
- **核心功能**:
  - **Codex**: 自动追踪系统
    - 提及追踪：自动标记角色/地点出现
    - 进展追踪：故事进度
    - 别名系统：角色多名字管理
    - 标签分类
    - 关系网络
    - 时间线
  - **300+模型BYOK**: 自带API Key，支持Claude/GPT/Gemini/本地模型
  - **Plan**: 故事规划工具
  - **Matrix**: 多维度故事结构矩阵
  - **Scene Labels**: 场景标签分类
  - **Tinker Chat**: AI对话式写作助手
  - **模板**: Save the Cat / Hero's Journey / 3-Act Structure
- **长篇处理方式**:
  - Codex自动维护所有追踪信息
  - 场景级生成，Codex自动注入相关上下文
  - 支持BYOK意味着可选最大上下文模型
  - 手动管理场景顺序和结构
- **用户评价**:
  - 最佳性价比（$4起+自带Key）
  - Codex系统非常强大
  - 界面美观，使用流畅
  - 社区活跃，更新频繁
  - 缺点：需要自己有API Key

### 3. NovelAI

- **URL**: https://novelai.net
- **定价**: $10/月（Tablet），$15/月（Scroll），$25/月（Opus）
- **核心功能**:
  - **Lorebook**: 关键词触发注入系统
    - 设置关键词，当文本中出现该词时自动注入相关信息
    - 支持正则表达式匹配
    - 优先级控制
    - 嵌套和分组
  - **Memory**: 持久字段，始终在上下文中
  - **Author's Note**: 风格/语调指令，高优先级注入
  - **Genji**: 专门训练的日语模型
  - **加密存储**: 端到端加密，隐私保护
  - **专训模型**: Kayra / Euterpe / Krake 等
  - **图像生成**: 内置AI绘图（动漫风格）
- **长篇处理方式**:
  - Lorebook关键词触发自动注入相关世界观
  - Memory持久上下文约束全局设定
  - 手动管理记忆窗口
  - 适合互动式写作（非一键生成）
- **用户评价**:
  - 隐私保护最好
  - 日语/动漫向最强
  - 互动式写作体验优秀
  - 不适合想要自动化的用户
  - Lorebook系统学习曲线陡峭

### 4. Squibler

- **URL**: https://squibler.io
- **定价**: $16/月（Basic），$33/月（Standard），$49/月（Premium）
- **核心功能**:
  - **从概念直接生成整书**: 输入想法即可生成完整书稿
  - **Elements系统**: 维护项目智能（角色/设定/情节元素）
  - **80+语言翻译**: 自动翻译输出
  - **视觉生成**: 角色/场景可视化
  - **协作功能**: 多人协作写作
  - **模板库**: 多种类型模板
- **长篇处理方式**:
  - Elements系统维护全局一致性
  - 分章节生成
  - 自动翻译扩展受众
- **用户评价**:
  - 适合快速出书
  - 质量不如Sudowrite
  - 翻译功能实用
  - 适合非虚构类更好

### 5. Subtxt

- **URL**: https://subtxt.app
- **定价**: $20/月（Basic），$50/月（Pro），$250/月（Enterprise）
- **核心功能**:
  - **32000+叙事结构**: 基于Dramatica理论的巨大结构库
  - **Dramatica理论**: 完整实现Dramatica叙事理论
  - **四条故事线**: Main Character / Influence Character / Relationship / Overall Story
  - **实时主题分析**: 追踪主题发展
  - **Emergent Story Fields**: AI驱动的故事结构涌现
  - **Muse AI**: 对话式创意伙伴
- **长篇处理方式**:
  - 基于理论的结构化规划
  - 四条故事线并行管理
  - 32000+结构模式可选
  - 更侧重规划而非生成
- **用户评价**:
  - 叙事理论最深的工具
  - 学习曲线极陡峭
  - 适合编剧和严肃作家
  - 不适合想要快速出文的用户

### 6. LivingWriter

- **URL**: https://livingwriter.com
- **定价**: 免费起，$9.99/月（Pro），$19.99/月（Pro+AI）
- **核心功能**:
  - **AI大纲**: 自动生成故事大纲
  - **AI元素生成**: 角色/地点/物品自动生成
  - **AI重写**: 段落/章节级重写
  - **AI手稿对话**: 与手稿对话获取建议
  - **AI分析**: 故事分析和建议
  - **AI封面**: 自动生成封面
  - **元素追踪**: 自动追踪角色/地点出现
- **长篇处理方式**:
  - 元素追踪维护一致性
  - 大纲→章节逐步生成
  - AI对话式引导写作
- **用户评价**:
  - 界面友好
  - 免费版功能有限但够用
  - 适合入门作家
  - AI质量中等

### 7. AutoCrit

- **URL**: https://www.autocrit.com
- **定价**: 免费起，$30/月（Professional），$80/月（Fiction）
- **核心功能**:
  - **20+编辑工具**: 节奏/对话/词汇/句式等分析
  - **系列分析器**: 跨书一致性检查
  - **AI读者档案**: 模拟不同读者的反馈
  - **类型对标基准**: 与同类型畅销书对比
  - **强词/弱词检测**: 优化文字力度
  - **重复检测**: 词/短语/句式重复
  - **节奏分析**: 句长变化可视化
- **长篇处理方式**:
  - 偏编辑/修订而非生成
  - 系列分析器处理多本书一致性
  - 类型基准帮助对标市场
- **用户评价**:
  - 编辑功能强大
  - 不生成内容，只分析和建议
  - 适合已完成初稿的修改阶段
  - 系列分析器对长篇系列作者很有价值

### 8. Fictionary (StoryTeller)

- **URL**: https://fictionary.co
- **定价**: 免费试用，$20-50/月
- **核心功能**:
  - **38个小说写作元素分析**: 覆盖故事的所有关键维度
  - **场景结构分析**: 每个场景的入口/目标/冲突/结果
  - **角色弧追踪**: 角色发展轨迹
  - **节奏分析**: 故事节奏图表
  - **类型指南**: 不同类型的要素清单
- **长篇处理方式**:
  - 38元素体系化分析
  - 场景级颗粒度
  - 可视化角色弧和情节弧
- **用户评价**:
  - 分析最系统化
  - 适合结构化修改
  - 不生成内容
  - StoryTeller版本加入了AI建议

## 中文市场

### 1. 阅文妙笔

- **URL**: 阅文集团内部工具/合作作者使用
- **定价**: 合作作者免费使用
- **核心功能**:
  - 中国最大网文平台（起点中文网）集成
  - 理解网文类型惯例（玄幻/都市/修仙等）
  - 集成发布/变现管道
  - AI续写和大纲建议
  - 角色设定管理
- **长篇处理方式**:
  - 平台数据训练，理解网文节奏
  - 大纲→章节引导式生成
  - 与发布流程深度集成
- **用户评价**:
  - 与起点生态深度绑定
  - 网文类型理解最好
  - 非公开服务，需要合作资格
  - 质量不稳定

### 2. 秘塔写作猫 (xiezuocat.com)

- **URL**: https://xiezuocat.com
- **定价**: 免费基础版，会员制
- **核心功能**:
  - AI写作助手：续写/扩写/改写
  - 语法风格校正
  - 内容扩展重写
  - 多种写作模式
  - 查重功能
- **长篇处理方式**:
  - 段落级辅助为主
  - 语法和风格校正
  - 不是专门的长篇工具
- **用户评价**:
  - 中文语法校正最好
  - 创意生成一般
  - 适合学术/公文/日常写作
  - 长篇小说不是主要场景

### 3. 火山写作

- **URL**: 字节跳动产品
- **定价**: 免费/会员制
- **核心功能**:
  - 字节跳动生态
  - 豆包大模型驱动
  - AI续写/改写/翻译
  - 多场景模板
- **长篇处理方式**:
  - 段落级辅助
  - 模板引导
  - 与字节生态（番茄小说）可能有联动
- **用户评价**:
  - 依托豆包模型，中文能力强
  - 生态联动潜力
  - 目前更偏通用写作

### 4. 笔灵AI (ibiling.cn)

- **URL**: https://ibiling.cn
- **定价**: 免费体验，会员制
- **核心功能**:
  - **200+写作工具**: 覆盖各种写作场景
  - **黄金开头生成**: 专门优化小说开头
  - **编辑认证大纲**: AI生成大纲经编辑认证
  - **降AI痕迹**: 专门的去AI味功能
  - 多种小说类型支持
  - 章节生成+润色
- **长篇处理方式**:
  - 大纲→章节生成流程
  - 黄金开头专项优化
  - 去AI味后处理
  - 200+工具组合使用
- **用户评价**:
  - 功能丰富
  - 去AI味效果受好评
  - 黄金三章优化实用
  - 质量中上

### 5. 蛙蛙写作 (WawaWriter)

- **URL**: https://www.wawawriter.com
- **定价**: 免费体验，会员制
- **核心功能**:
  - **完整网文平台**: 从构思到发布
  - **短篇过稿**: 专门的短篇小说创作流程
  - **长篇流程**: 大纲→章节→润色→去AI味
  - 类型模板（玄幻/都市/言情等）
  - 多风格选择
  - 一键去AI味
- **长篇处理方式**:
  - 结构化长篇流程：
    1. 类型选择
    2. 大纲生成
    3. 章节扩写
    4. 润色修改
    5. 去AI味处理
  - 短篇可直接过稿到平台
- **用户评价**:
  - 专为中文网文设计
  - 短篇过稿率较高
  - 长篇需要人工干预
  - 去AI味是亮点

### 6. 非凡写作

- **URL**: https://feifanxiezuo.com
- **定价**: 会员制
- **核心功能**:
  - **扫榜**: 分析热门作品数据
  - **对标**: 找到对标作品分析结构
  - **细纲**: 详细章节大纲生成
  - **正文**: 基于细纲生成正文
  - 专业生产流程
  - 数据驱动选题
- **长篇处理方式**:
  - 扫榜→对标→细纲→正文 四步流程
  - 数据驱动：基于市场数据选择方向
  - 对标学习：分析成功作品结构
  - 细纲到正文的结构化生成
- **用户评价**:
  - 最专业的网文生产工具
  - 扫榜功能实用
  - B站有详细教程（32.2万播放）
  - 适合职业网文作者

## 已停服产品

### 彩云小梦 Dreamily.ai

- **状态**: 2026年4月21日正式停服
- **数据**: 已永久删除
- **历史定位**: 
  - 中国最早的AI小说生成平台之一
  - 互动式写作体验
  - 多种类型支持
  - 社区创作分享
- **停服原因**: 未公开，可能与商业化困难和监管环境有关
- **教训**: 依赖单一平台的写作数据风险
