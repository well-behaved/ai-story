# ai-story

面向番茄小说的 Claude Code 写作 Skill，支持 1000+ 章超长篇网文全流程 AI 辅助创作。

## 功能特性

- **5问快速开书** — 题材/主角/冲突/金手指/基调，5个问题开始写作
- **四级大纲体系** — 总纲 → 卷纲 → 章纲，分卷管理不怕长篇
- **三智能体写作循环** — Writer 写 → Reviewer 审(10维50分) → Polisher 润(六大门禁去AI味)
- **分层文件记忆** — P0-P5 优先级上下文组装，第1章和第1000章注入量一样
- **伏笔自动追踪** — 紧迫度分级(潜伏→铺垫→可回收→紧急回收)，到期自动提醒
- **分层摘要压缩** — 近10章详细/11-50中等/51+一句话/卷级摘要，永不膨胀
- **6维全局审查** — 角色偏移/力量自洽/时间线/伏笔回收/节奏分布/情节线
- **六大门禁去AI味** — 禁用词→句式打散→心理外化→节奏打碎→对话去腔调→结尾去升华
- **番茄平台适配** — 5种题材模板 + 敏感词检查 + 番茄格式导出 + 黄金三章专项审查

```bash

git clone https://github.com/well-behaved/ai-story.git ~/.claude/skills/ai-story

# 方式二：直接在项目目录使用
git clone https://github.com/well-behaved/ai-story.git
```

### 使用

```
你：帮我开一本小说
AI：好的，先回答5个问题...
    Q1: 什么题材？（系统文/升级文/重生文/都市文/赘婿文）
    Q2: 主角什么样？
    Q3: 核心冲突？
    Q4: 金手指/系统？
    Q5: 什么基调？

你：写大纲
AI：生成总纲 → 卷纲 → 章纲...

你：日更
AI：组装上下文 → 写初稿 → 10维审查(45/50) → 六大门禁润色 → 更新追踪 → 完成
    ✅ 第1章 已完成，2600字，审查45/50，润色2处

你：继续
AI：写第2章...

你：去AI味
AI：六大门禁处理，修改12处，修改率8%

你：全局审查
AI：角色一致:高 | 力量自洽:高 | 时间线:无矛盾 | 伏笔回收:85% | 节奏:正常
```

### 快速启动模式

## 写作流水线

每章经过完整的 7 阶段处理：

```
Stage 1: Planner 组装上下文（P0骨架 + P1角色 + P2前章 + P3伏笔 + P4摘要 + P5风格）
    ↓
Stage 2: Writer 写初稿（2000-4000字，遵循章纲+风格规范+禁用词）
    ↓
Stage 3: Reviewer 10维审查（>=35分通过，<35分返回重写，最多3轮）
    ↓
Stage 3.5: Polisher 六大门禁去AI味（修改量<=15%）
    ↓
Stage 4: 保存正文（先于追踪更新，确保不丢稿）
    ↓
Stage 5: Tracker 更新追踪（角色状态/伏笔/摘要/情节线/.story-state）
    ↓
Stage 6: 输出结果 + 自动触发检查（连续低分/定期审查/卷末提醒）
```

## 1000章怎么做到的

核心原则：**上下文大小恒定，不随章节数增长**。

| 层级 | 内容 | 大小 | 说明 |
|------|------|------|------|
| P0 | 设定+总纲摘要+卷纲+章纲 | ~3000字 | 只取核心主线，不取全部 |
| P1 | 本章出场角色状态 | ~1500字 | 按需加载，不全量 |
| P2 | 前1章完整正文 | ~2500字 | 衔接语感 |
| P3 | 活跃伏笔(前20条) | ~1000字 | 按紧迫度排序，硬上限 |
| P4 | 最近5章摘要 | ~1500字 | 从详细区取 |
| P5 | 风格规范 | ~500字 | 含前3章黄金三章约束 |
| **总计** | | **~10000字** | **第1章和第1000章一样** |

## 技术细节

- **记忆方案**：分层文件记忆（Markdown），不需要数据库
- **状态管理**：快照式（覆写当前值），不是追加日志
- **伏笔升级**：30%→铺垫 / 70%→可回收 / 120%→紧急回收 / 50章强制
- **摘要压缩**：近10章详细(250字) / 11-50中等(75字) / 51+一句话(20字) / 卷级(800字)
- **分卷策略**：每卷50-100章，切换时压缩旧卷+归档已回收伏笔+清理退场角色
- **编号策略**：卷内从001起，全书连续编号=前几卷章数之和+卷内章号
- **中断恢复**：writing_in_progress 标记6阶段状态，compact前自动快照
