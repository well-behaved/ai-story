# 章末钩子设计指导

> 加载时机：story-outline（章纲中标注钩子类型时）、story-write（写章末时参考）

---

## 一、9 种钩子类型

### 1. 问题钩
**原理**：提出一个读者迫切想知道答案的问题
**示例**：
```
那封信的最后一行只有四个字——
"你父未死。"
```
**适用**：揭秘类剧情、身世之谜、阴谋类

### 2. 揭示钩
**原理**：揭露一个震撼信息，然后立刻切断
**示例**：
```
面具摘下来的那一刻，张三的瞳孔骤缩。
那张脸——
他认识。
```
**适用**：反转、身份揭露、真相大白前

### 3. 危险钩
**原理**：主角陷入即时危险，生死未卜
**示例**：
```
脚下的地面开始碎裂，岩浆的热浪扑面而来。
退路已经没有了。
身后传来第二声咆哮。
```
**适用**：战斗、逃亡、险境、副本探索

### 4. 决定钩
**原理**：主角面临两难选择，切在做决定之前
**示例**：
```
左手边是通往逃生出口的通道。
右手边传来李四微弱的呼救声。
他只有三十秒。
```
**适用**：道德困境、策略选择、感情抉择

### 5. 到达钩
**原理**：新角色/新势力/新事件突然出现
**示例**：
```
大门被一脚踹开。
所有人的目光看向门口——
来的人，没有一个是他们想见到的。
```
**适用**：新角色登场、意外来客、事态升级

### 6. 承诺钩
**原理**：暗示即将发生的重大事件
**示例**：
```
老者把一枚令牌拍在桌上。
"三天后，天罗殿开门。"
"届时——"他看了张三一眼，"你会知道什么叫真正的修行。"
```
**适用**：新副本预告、大事件倒计时、实力展示预告

### 7. 反转钩
**原理**：读者以为的事实被推翻
**示例**：
```
"我已经查清楚了，杀你师父的人——"
王五顿了顿，看了看在场所有人。
"——就在这个房间里。"
```
**适用**：悬疑、背叛、阴谋、身份反转

### 8. 情感钩
**原理**：强烈情感状态未释放就切断
**示例**：
```
她转过身去。
"别跟来。"
声音在发抖，但脚步没有停。
张三站在原地，雨水打在脸上。他张了张嘴，什么也没说出来。
```
**适用**：感情线、离别、误会、牺牲

### 9. 时限钩
**原理**：倒计时开始，制造紧迫感
**示例**：
```
毒已经扩散到了心脉。
"最多七天。"老大夫合上药箱，摇了摇头。"七天之内找不到冰蚕草，神仙也救不了她。"
七天。
整座雪山只在最深处才有冰蚕草的记载。
```
**适用**：救人、倒计时任务、限时挑战

## 二、钩子力度等级

### 弱钩（维持阅读）
- 一个小问题未解答
- 一个小任务待完成
- 一句意味深长的话
- 适用：过渡章、日常章

### 中钩（驱动翻页）
- 新信息暴露但未展开
- 中等危险出现
- 关系变化暗示
- 适用：铺垫章、发展章

### 强钩（不看下一章睡不着）
- 生死悬念
- 重大反转
- 核心真相揭露一半
- 适用：高潮前夜、重要转折、卷末

## 三、钩子分布策略

### 推荐节奏：2弱 → 1中 → 1强

```
第1章：弱钩（信息暗示）
第2章：弱钩（小悬念）
第3章：中钩（新角色/新威胁）
第4章：强钩（危险/反转）
第5章：弱钩（缓一下）
第6章：中钩（情感/决定）
第7章：弱钩
第8章：强钩（副本高潮前）
...
```

### 关键位置必须用强钩
- 每卷第 1 章（抓住读者）
- 每个副本开始
- 副本高潮前一章
- 卷末最后一章

### 不要每章都用强钩
- 读者会疲劳
- 强钩太多等于没有强钩
- 弱钩和中钩制造"呼吸感"

## 四、钩子与下一章开头的衔接

### 错误做法：立刻化解

```
第10章末：张三中了剧毒，命悬一线——
第11章开头：幸好这时候王五及时赶到，一颗解毒丹救了他。
```
→ 读者感觉被欺骗，钩子没有意义。

### 正确做法：延迟回应

**方法 A：切视角**
```
第10章末：张三中了剧毒——
第11章开头：（切到另一个角色的视角，处理其他事情）
第11章末或第12章：回到张三的剧毒危机
```

**方法 B：加码**
```
第10章末：张三中了剧毒——
第11章开头：毒比他想象的更严重，而且解药只在敌人手里
```
→ 不化解悬念，而是让悬念升级。

**方法 C：部分回应**
```
第10章末：面具下是谁？——
第11章开头：张三认出了这张脸，但更大的问题是——他为什么会出现在这里？
```
→ 回答了"是谁"，但抛出了更大的"为什么"。

## 五、各题材常用钩子组合

### 系统文
- 常用：承诺钩（系统发布新任务）+ 时限钩（任务倒计时）
- 升级关键时刻用危险钩

### 升级文/修仙
- 常用：危险钩（强敌出现）+ 揭示钩（秘境发现）
- 突破时用情感钩+到达钩

### 重生文
- 常用：反转钩（利用前世记忆反杀）+ 揭示钩（提前知道真相）
- 前世仇人出现用到达钩

### 都市文
- 常用：到达钩（新势力介入）+ 决定钩（商业/感情抉择）
- 身份暴露用揭示钩

### 悬疑/推理
- 常用：问题钩（新线索）+ 反转钩（嫌疑人反转）
- 真相揭露前用揭示钩
