# 副本/剧情弧线结构

> 加载时机：story-outline（规划章纲时）、story-write（写副本章节时参考节奏）

---

## 一、标准副本 7 阶段

每个副本（剧情弧线）遵循 7 阶段结构：

| 阶段 | 名称 | 章节占比 | 核心任务 |
|------|------|---------|---------|
| 1 | 入场 | 10% (1-2章) | 新环境、新规则、新目标 |
| 2 | 探索 | 15% (2-3章) | 了解环境、结交盟友、发现线索 |
| 3 | 小冲突 | 25% (3-5章) | 小Boss/障碍、展示能力、建立威望 |
| 4 | 受挫 | 10% (1-2章) | 遭遇真正强敌、计划失败、盟友背叛 |
| 5 | 成长 | 15% (2-3章) | 修炼/策划/获得关键信息或道具 |
| 6 | 高潮 | 15% (2-3章) | 大Boss决战、全力爆发、逆转翻盘 |
| 7 | 过渡 | 10% (1-2章) | 收获奖励、关系推进、下一副本伏笔 |

---

## 二、每阶段详细写法

### 阶段 1：入场（1-2 章）

**目标**：建立副本的"游戏规则"

**要点**：
- 主角为什么来这里？（目标驱动：寻宝/历练/报仇/救人/被迫）
- 这个环境的特殊规则是什么？（副本比外部世界更危险/更有机遇）
- 第一印象冲击（让读者感受到"不一样了"）

**钩子**：入场章末尾必须暗示危险或机遇

**模板**：
```
主角到达新地点 → 见识到环境的独特/危险 → 遇到第一个NPC（盟友or向导）
→ 得知副本的核心规则/目标 → 章末：暗示更深处有更大的秘密/危险
```

### 阶段 2：探索（2-3 章）

**目标**：扩展信息量，为后续冲突铺垫

**要点**：
- 主角适应环境，展示聪明/观察力
- 结交 1-2 个副本专属配角（后续有用）
- 发现副本内的势力格局/潜规则
- 小试牛刀：解决一个小问题获得初步认可

**节奏**：相对平缓，但每章至少有一个信息增量

### 阶段 3：小冲突（3-5 章，爽点密集区）

**目标**：制造连续爽点

**要点**：
- 遇到副本内的"小反派"（嚣张、瞧不起主角）
- 打脸 × 2-3 次（逐步升级对手难度）
- 展示主角的特色能力
- 获得一些资源/奖励
- 旁观者从质疑到惊叹

**爽点分布**：每章至少一个打脸/获宝/展示

### 阶段 4：受挫（1-2 章，情绪低谷）

**目标**：制造紧张感和悬念

**要点**：
- 出现真正的强敌（不是之前的小喽啰级别）
- 主角初次交锋失利或被算计
- 盟友可能被威胁/受伤/背叛
- 主角认识到自己还不够强

**关键**：受挫要真实但不绝望。读者要担心但不放弃希望。

**禁忌**：不要让主角被暴打得太惨（读者会不爽），轻伤+战略失败即可。

### 阶段 5：成长/准备（2-3 章）

**目标**：为高潮积蓄力量

**要点**：
- 获得关键道具/信息/盟友
- 可能有一次小突破（升级0.5-1个小等级）
- 制定对付大Boss的计划
- 与盟友加深关系（为高潮的配合做铺垫）

**节奏**：不能太慢（读者等不及），也不能太快（显得太容易）。2-3 章为佳。

### 阶段 6：高潮（2-3 章，最大爽点）

**目标**：释放积累的所有张力

**写法模板**：
```
第1章：决战开始
  - 与大Boss正面对决
  - 先占上风（给读者希望）→ 被反击（紧张）
  - 章末：危急时刻

第2章：逆转翻盘
  - 使用准备阶段获得的关键道具/信息
  - 或者突破（战斗中突破是最燃的）
  - 全力爆发，击败大Boss
  - 旁观者震惊/欢呼

第3章（可选）：连锁反应
  - 如果是大副本，击败Boss后还有连锁事件
  - 真相揭露/更大阴谋浮现/意外发现
```

**爽点**：这里是全副本的爽点峰值，不要吝啬。打脸+升级+逆转叠加。

### 阶段 7：过渡（1-2 章）

**目标**：收束本副本，铺垫下一个

**要点**：
- 收获奖励（具体化：等级提升/宝物/名望/关系）
- 副本内配角的去向（分别/加入/牺牲）
- 解决一个悬念，留下一个新悬念
- 暗示下一个更大的挑战

**钩子**：过渡章末必须有指向下一副本的强钩子

## 三、不同规模的副本

### 小副本（5-8 章）
- 适用：日常任务、小型历练、单次事件
- 简化：合并阶段 2+3（探索和小冲突一起），省略阶段 5（直接从受挫到高潮）
- 结构：入场(1) → 探索+冲突(2-3) → 受挫(1) → 高潮(1-2) → 过渡(1)

### 中副本（10-15 章，最常用）
- 适用：标准剧情弧线
- 完整 7 阶段
- 结构：入场(1-2) → 探索(2-3) → 小冲突(3-4) → 受挫(1-2) → 成长(2) → 高潮(2-3) → 过渡(1-2)

### 大副本（15-25 章）
- 适用：卷级高潮、大型秘境、重要剧情转折
- 扩展阶段 3 和 6，可能有多波小冲突和多轮高潮
- 结构：入场(2) → 探索(3) → 小冲突×2波(5-6) → 受挫(2) → 成长(3) → 高潮×2波(4-5) → 过渡(2)

## 四、副本之间的过渡技巧

### 1. 无缝衔接
上一副本的过渡章直接包含下一副本的触发事件。
例：刚从秘境出来，发现宗门被袭击了 → 直接进入新副本。

### 2. 短暂休整
1-2 章日常/休息/整理收获，然后新事件发生。
适用于读者需要喘口气的时候。不要超过 2 章（会掉读者）。

### 3. 时间跳跃
"三个月后——" 跳过平淡期，直接到下一个有事发生的时间点。
适用于需要时间流逝但没有值得写的事件时。

### 4. 交叉铺垫
在副本 A 的中后期就开始铺垫副本 B 的信息/角色，过渡时读者已经有期待。

## 五、副本升级原则

每个新副本必须在以下至少 2 个维度上"升级"：

1. **敌人更强**：新Boss比上一个更难对付
2. **规模更大**：从个人恩怨→组织冲突→国家级事件
3. **情感更深**：主角的个人利益卷入更多（不只是打怪升级）
4. **真相更近**：全书主线的谜团揭露更多
5. **代价更高**：可能有牺牲/失去/不可逆的损失
