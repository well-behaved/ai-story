# 重生文写作模板

> **加载时机**：当 story-start 流程中用户选择"重生文""穿越重生"题材，或 AI 判断故事构思包含"回到过去""带着记忆重来"等重生元素时，加载本模板作为写作指导。

---

## 1. 题材定义与核心卖点

重生文的核心设定是：主角因某种原因（惨死、遗憾、意外）带着前世的完整或部分记忆回到过去某个时间节点，利用信息差重新来过，弥补遗憾、复仇翻盘。

**读者为什么爱看：**
- **"如果能重来"的终极幻想**：每个人都有遗憾，重生文直击这个人类共通的愿望
- **信息差带来的智商碾压**：主角知道未来，提前布局，读者跟着一起"上帝视角"操盘
- **打脸的极致快感**：前世害过主角的人，这一世被一一清算，复仇爽感拉满
- **"早知如此"的代入感**：前世错过的人、错过的机会，这一世全部抓住
- **悬念天然存在**：蝴蝶效应——主角的改变会让未来偏离轨迹，读者好奇"还能按剧本走吗"

**数据表现**：重生文在番茄属于高开高走型，开篇数据极好（前世惨+重生逆袭的设定天然吸引人），中期保持力也强（信息差红利持续释放）。男频女频都有大量受众。

---

## 2. 典型世界观框架

### 时间框架

重生文的世界观核心是"两条时间线"：

**前世时间线**（回忆线）：
- 主角的完整人生经历，从起点到死亡
- 关键节点清单：重大事件、重大决策、遗憾的转折点
- 记忆不是完美的：大事记得清楚，小细节模糊，这很重要

**今生时间线**（主线）：
- 重生回到的时间点（通常是人生的某个关键转折前）
- 从这个节点开始，主角做出不同选择
- 随着改变积累，偏离前世轨迹越来越多

### 重生设定三要素

**1. 重生原因**（选一种）：
- 惨死型：被害/被背叛/含恨而死 → 动力是复仇+自保
- 遗憾型：人生充满错误选择 → 动力是弥补遗憾
- 意外型：不明原因重生 → 动力是珍惜这次机会

**2. 重生时间点**（选一种）：
- 高中/大学时期：青春感强，成长空间大，适合都市文
- 刚入职/创业初期：商战为主，适合商业文
- 重大灾难/变故前夕：紧迫感强，适合末日/异能文
- 修炼关键节点：适合玄幻/修仙重生文

**3. 记忆范围**：
- 大事件：清晰记得（股市走向、重大灾难、关键人物命运）
- 中等事件：部分记得（某些细节模糊，需要推理）
- 小事件：基本不记得（日常对话、具体日期可能记不清）
- 这个分层很关键——全知全能的重生者没有看点，有模糊地带才有悬念

### 蝴蝶效应机制

主角的每一个改变都会产生涟漪：
- **短期影响**（1-20章）：微小变化，前世记忆依然高度可靠
- **中期影响**（20-80章）：部分事件时间和细节开始偏移
- **长期影响**（80章+）：前世记忆的"保质期"逐渐到期，主角必须靠自身能力面对全新局面

## 3. 典型角色配置

### 主角（重生者）
- **前世形象**：失败者/受害者/被背叛者（前世越惨，重生越有动力）
- **今生性格变化**：相比前世更果断、更冷静、更有大局观，但偶尔流露超越年龄的成熟（被人注意到的伏笔）
- **核心矛盾**：利用前世信息到什么程度？是完全功利地利用，还是真心对待身边的人？
- **成长弧线**：从"依赖前世记忆"到"凭借自身能力开创全新未来"

### 前世仇人
- **身份**：前世害主角最深的人（背叛的妻子/陷害的兄弟/霸占成果的合伙人）
- **今生处理**：不是一上来就报仇，而是布局让对方一步步走入陷阱
- **要有层次感**：小仇小恨先清算（3-10章解决），大仇大恨要铺长线（50-100章）
- **反派不能太蠢**：前世能害到主角说明有能力，今生也不能变成傻子

### 前世错过的人
- **身份**：前世没珍惜的恋人/没帮助的恩人/没结交的贵人
- **今生处理**：主角主动接近、珍惜、帮助——但不能表现得太刻意，否则会引起怀疑
- **矛盾点**：对方并不知道前世的缘分，主角的"莫名关心"可能让对方困惑
- **感情线价值**：重生文的感情线天然有深度——"你不知道，我已经爱了你两辈子"

### 前世的贵人/导师
- **前世关系**：曾经帮助过主角但主角没来得及报答（或贵人后来遭难了）
- **今生处理**：提前找到并帮助，形成更深的联盟
- **作用**：提供主角凭记忆无法获得的资源和人脉

### "发现者"（可选但推荐）
- 某个特别敏锐的人注意到主角"不对劲"——言行举止超越年龄、对未来事件的精准预判
- 制造紧张感：主角的秘密有被揭穿的风险
- 可以是盟友（发现后选择帮忙），也可以是威胁（试图利用主角的能力）

## 4. 典型情节结构

### 开局（第1-3章）：重生降临

**关键原则：第一段就交代"重生了"，不要花几章铺垫。**

```
第1章开头：前世的最后时刻（惨死/绝望的场景，200-300字快速交代）
         → 猛然睁眼，发现回到了过去（明确交代时间和场景）
         → 确认重生（看日历/照镜子/确认周围人），内心独白（前世仇恨/遗憾）
第2章：前世的第一个关键人物出现（仇人或贵人），主角做出与前世不同的选择
第3章：第一波逆袭——利用前世记忆获得第一个优势（赚到第一笔钱/避开第一个陷阱/获得第一个机会）
```

### 发展（第4-80章）：信息差红利期

**短线复仇弧（每5-10章一个）：**
```
前世仇人出场 → 主角提前知道对方的计划/弱点 → 布局设套
→ 仇人掉入陷阱 → 旁观者震惊于主角的"先见之明" → 清算完毕
```

**中线布局弧（每15-25章一个）：**
```
主角利用前世记忆提前布局（投资/站队/结交关键人物）
→ 众人不理解主角为什么这么做 → 事件按主角预判发展
→ 众人震惊："他怎么知道？！" → 主角地位/资源大幅提升
```

**感情线弧（贯穿全文）：**
```
与前世错过的人重新相遇 → 主角刻意接近但装作自然
→ 相处中感情升温 → 遇到前世曾导致分离的事件 → 主角改变结局
→ 感情深化，但主角内心的秘密成为隐患
```

### 转折（第40-80章）：蝴蝶效应期

```
前世的某个"必然事件"没有发生（因为主角的改变）
→ 主角开始发现前世记忆不再可靠 → 新的未知局面出现
→ 主角必须依靠自身能力而非信息差来应对 → 真正的成长开始
```

### 高潮（每80-120章一个大高潮）

```
大仇人的终极计划（前世中成功的阴谋） → 主角提前知道大框架但细节已经改变
→ 敌我双方都在博弈 → 主角利用"半知半不知"的信息打出神操作
→ 大仇人落败 → 更大的格局展开
```

## 5. 典型爽点类型和节奏

### 核心爽点清单

| 爽点类型 | 频率 | 说明 |
|----------|------|------|
| 信息差打脸 | 每3-5章 | "他怎么知道？"类反应，核心爽点 |
| 复仇清算 | 每5-10章 | 前世仇人被一个个清算 |
| 提前布局成功 | 每8-15章 | 众人不理解→事后证明主角是对的 |
| 弥补前世遗憾 | 每10-20章 | 救下前世死去的人/抓住错过的机会 |
| 身份/实力暴露 | 每15-25章 | 主角隐藏的能力/财富/地位被发现 |
| 蝴蝶效应意外 | 每20-30章 | 前世没发生的新事件，增加不确定性 |

### 回忆与现实的节奏

- **前世回忆不要大段插叙**，用斜体或单独短段标注（3-5行以内）
- **回忆穿插在关键决策前**："上一世，这个时候他做了XX选择，结果......这一次，我不会再犯同样的错。"
- **前世回忆占比控制在10-15%**，太多会打断现实剧情的节奏
- **回忆要有情绪**，不是干巴巴地回顾事件，而是带着后悔/愤怒/感慨的情绪

### 悬念节奏

- 每5章要制造一个"前世没发生过的意外"，提醒读者和主角：这一世不是简单复读
- 每10-15章要有一个"主角差点暴露重生者身份"的险境
- 每20-30章要有一次"前世记忆彻底失效"的情况，倒逼主角靠自己

## 6. 金手指/系统设计建议

### 重生文的"金手指"就是前世记忆，但可以叠加辅助金手指

**记忆本身的设计：**
- 大事件记忆清晰（重大转折、灾难、风口）
- 专业知识保留（前世的专业技能、人脉信息）
- 细节逐渐模糊（具体日期、对话内容可能不精确）
- 超过前世死亡时间点后完全无效（终极时间限制）

**可叠加的辅助金手指（选一个，不要贪多）：**
- **神秘空间**：重生时带回的空间（存储前世带回的物品/书籍）
- **身体强化**：重生后体质变异，修炼速度加快
- **感知增强**：对危险的直觉预警（前世死过一次的人对危险更敏感）
- **前世技能保留**：前世的武功/医术/厨艺等技能直接保留在身体记忆中

### 金手指衰减机制

重生文最怕写成"开挂无敌"，必须设计衰减：
1. **记忆衰减**：越偏离前世轨迹，记忆越不可靠
2. **时间限制**：超过前世死亡时间后，完全没有信息优势
3. **蝴蝶效应加速**：主角改变越多，未来偏差越大
4. **心理负担**：前世记忆带来的创伤/压力影响判断

## 7. 番茄平台注意事项

### 展示规范
- 前世回忆用**斜体**或**单独段落**标注，与现实剧情明确区分
- 不要用大段倒叙来回顾前世，用碎片化的方式穿插
- 时间线要清晰，每到新的时间节点要明确标注（"重生后第XX天"/"20XX年X月"）
- 内心独白适量，不要每章都大段心理活动

### 数据建议
- 重生文开局数据通常很好（设定天然吸引人），关键是中期保持
- 第1章必须完成"死亡→重生→确认时间"的全流程
- 前3章必须至少有一次信息差打脸
- 每章字数建议2000-2500字
- 章末钩子用"前世的某个关键人物即将出现"或"前世的某个大事件即将发生"效果最好

### 合规注意
- 前世的死法不要太血腥详细
- 不要涉及真实的历史事件预言（尤其是灾难、政治事件）
- 股市/彩票等"稳赚不赔"的剧情可以有，但不要写成教程
- 复仇手段不要违法犯罪（可以用商业手段、社交手段替代暴力）

---

## 8. 避免的陷阱

### 常见写崩原因

1. **开局太慢**：花了3-5章才重生 → 第1段就要交代重生，前世用几百字交代即可
2. **开挂感太强**：所有事都按前世记忆来，毫无悬念 → 要有蝴蝶效应，要有记忆模糊的地方
3. **大段回忆插叙**：动不动就回忆前世半章 → 回忆用碎片化的方式，3-5行搞定
4. **复仇太急**：前5章就把大仇人干掉了 → 复仇要有节奏，小仇先报，大仇慢慢来
5. **感情线太快**：因为前世认识就直接表白 → 今生的对方并不认识你，要重新建立关系
6. **忽略蝴蝶效应**：改变了很多事但世界线完全不变 → 一定要有连锁反应
7. **前世人设崩**：前世是废柴，重生后突然变成社交达人/商业天才 → 性格改变需要合理过渡，可以用"看透了人心""不再犯同样的错"来解释
8. **时间线混乱**：读者搞不清现在是什么时候 → 每章开头或关键情节要有时间锚点
9. **记忆设定前后矛盾**：前面说某些事记不清，后面又详细说出来了 → 维护一个"主角记忆清单"，记清楚什么记得什么不记得

### 自检清单

写作时定期检查：
- [ ] 前世回忆是否控制在合理篇幅内（不超过15%）？
- [ ] 最近5章是否有利用信息差的爽点？
- [ ] 蝴蝶效应是否在累积（世界线是否在偏离）？
- [ ] 主角是否在"依赖记忆"和"自主决策"之间有良好平衡？
- [ ] 前世记忆的设定是否前后一致？
- [ ] 时间线是否清晰可追踪？
