# 都市文写作模板

> **加载时机**：当 story-start 流程中用户选择"都市文""现代文"题材，或 AI 判断故事构思设定在现代都市环境（含医生文、兵王文、透视文、神豪文、直播文等子类型）时，加载本模板作为写作指导。

---

## 1. 题材定义与核心卖点

都市文设定在现代都市，主角在当代社会中拥有某种超越常人的能力或身份，从社会底层或被人忽视的位置开始崛起。这是代入感最强的题材——故事就发生在读者熟悉的现实世界里。

**读者为什么爱看：**
- **极强的代入感**：公司、学校、医院、饭局——这些场景读者天天经历，主角就像"开了挂的自己"
- **装逼打脸的极致快感**：被人看不起→亮出实力/身份→所有人震惊，这个模式在都市文中最有杀伤力
- **逆袭阶层的幻想**：从底层到顶层，从被鄙视到被仰望，直击读者的社会焦虑
- **美人如云的满足感**：都市文的女性角色最丰富（校花、总裁女、白富美、青梅竹马），感情线天然好看
- **贴近现实的爽感**：不需要理解修仙体系，直接就能感受到"有钱有权有实力"的快感

**数据表现**：都市文在番茄读者基数极大，尤其是25-40岁男性读者。完读率高，付费转化好。子类型丰富，竞争也很激烈，差异化很重要。

## 2. 典型世界观框架

### 现实世界 + 隐藏层

都市文的世界观是"95%的现实 + 5%的超越"：

**现实层**：
- 正常的现代社会（公司、学校、医院、商场、饭局、KTV）
- 正常的社会阶层（穷人/中产/富豪/顶级权贵）
- 正常的社会规则（赚钱、升职、创业、社交）

**隐藏层**（根据子类型选择）：
| 子类型 | 隐藏层 | 金手指 |
|--------|--------|--------|
| 医生文 | 医术通神 | 神医传承/异能诊断 |
| 兵王文 | 军方暗部 | 退伍特种兵/龙组成员 |
| 透视文 | 超凡异能 | 透视眼/预知/念力 |
| 神豪文 | 无限金钱 | 神秘系统/家族财富 |
| 直播文 | 网红经济 | 直播系统/超强运气 |
| 鉴宝文 | 古玩圈子 | 鉴宝之眼/大师传承 |

### 社会层级框架

都市文的"升级体系"就是社会阶层：

```
底层（打工/学生/无业） → 小有资产（第一桶金/小团队） → 行业精英（公司老板/名医/高手）
→ 城市大佬（上层社会/圈子核心） → 省级/国家级人物（顶级权贵/隐藏家族） → 国际级
```

每一层有对应的场景和人物群：
- 底层：出租屋、食堂、小公司、势利亲戚
- 中层：高档写字楼、酒店、商务饭局
- 上层：私人会所、别墅区、拍卖会、顶级医院VIP
- 顶层：家族宅邸、私人岛屿、军方基地

## 3. 典型角色配置

### 主角
- **表面身份**：普通人/穷学生/小职员/退伍军人/上门女婿
- **隐藏身份**（至少一个）：隐藏家族继承人/退役特种兵/神医传人/异能觉醒者
- **性格**：低调内敛、不主动炫耀、被逼急了才亮实力、有底线讲义气
- **核心魅力**：在被人轻视时不卑不亢，在展示实力时云淡风轻

### 女主/女性角色群（都市文感情线非常重要）
- **冷傲校花/总裁**：初期对主角冷淡甚至看不起，逐渐被主角能力折服
- **青梅竹马/邻家女孩**：从小认识，是主角最温暖的港湾
- **职场丽人**：精明干练，和主角在事业上互相帮助
- **注意**：感情线要有推进但不能太快，"追到手"的过程比"在一起"之后更好看

### 势利配角群（打脸对象）
- **势利亲戚**：看不起主角，在家庭聚会上冷嘲热讽
- **嚣张富二代**：有钱有势，处处针对主角，争女主
- **油腻上司/同事**：职场中使绊子、抢功劳
- **前女友/前女友现男友**：悔恨当初没眼力

### 大佬/伯乐
- **身份**：行业大佬/隐藏世家家主/军方高层
- **作用**：认出主角的真实实力或身份，公开力挺
- **最佳出场方式**：在主角被一群人看不起时出现，毕恭毕敬叫一声"先生"/"师弟"/"少爷"，全场震惊

## 4. 典型情节结构

### 开局（第1-5章）：身份落差+第一波打脸

```
第1章：展示主角的"低姿态"（被看不起/被退婚/被嘲讽/被开除）
       同时暗示隐藏身份（一个神秘电话/一个旧物件/一段回忆）
第2章：遇到第一个冲突（被富二代侮辱/被势利亲戚鄙视/被老板为难）
第3章：主角被逼展示第一次实力/身份（出手治病/一个电话调动资源/出手教训人）
第4章：旁观者震惊，态度180度转变，打脸效果拉满
第5章：更大的舞台出现（被大佬注意/接到新的挑战/卷入更大的事件）
```

### 发展（第6-80章）：装逼打脸升级循环

**标准打脸弧线（每5-10章一个）：**
```
新场景/新人物出现 → 主角因外表/身份被轻视 → 冲突升级（对方嚣张+挑衅）
→ 主角淡定应对/展示某方面实力 → 旁观者震惊+态度反转
→ 被打脸者不服/找更强的后台 → 后台出场也被打脸 → 弧线完成
```

**事业线弧线（每15-20章一个）：**
```
商业机会出现 → 主角凭眼光/能力/资源入局 → 对手/竞争者不屑
→ 商业博弈过程 → 主角取得关键优势 → 对手败北/被迫让步
→ 主角财富/地位上升一个台阶
```

**感情线弧线（贯穿全文，每5-8章推进一步）：**
```
初遇/被误解 → 偶然帮助/展示魅力 → 产生好感但有障碍
→ 危机中保护/被感动 → 关系确认 → 外部干扰（情敌/家族反对）
→ 共同面对 → 感情升华
```

### 高潮（每50-80章一个大高潮）

**身份揭露型高潮：**
```
大型聚会/公开场合 → 主角被人当众侮辱/看不起
→ 关键人物（军方/顶级家族/行业领袖）出场
→ 揭露主角的真实身份/地位 → 全场震惊 → 之前所有看不起主角的人集体变脸
```

**危机逆转型高潮：**
```
主角或身边人遇到大危机（绑架/商业陷阱/生死危机）
→ 对手以为稳操胜券 → 主角动用隐藏实力/资源 → 逆转翻盘
→ 震撼全城/全行业
```

## 5. 典型爽点类型和节奏

### 核心爽点清单

| 爽点类型 | 频率 | 说明 |
|----------|------|------|
| 装逼打脸 | 每3-5章 | 被人看不起→亮实力→震惊，都市文灵魂 |
| 身份暴露 | 每10-15章 | 隐藏身份被部分揭露，态度反转 |
| 商业/事业成功 | 每8-12章 | 赚到大钱/签下大单/公司壮大 |
| 美人倾心 | 每5-10章 | 冷傲女神/校花被主角打动 |
| 大佬力挺 | 每10-15章 | 大人物公开为主角站台 |
| 秀技碾压 | 每5-8章 | 医术/武功/鉴宝等专业技能碾压 |
| 仇人落败 | 每15-20章 | 之前的对手被彻底击垮 |

### 打脸层级设计

打脸对象要有递进，不能一直打同级别的脸：
```
Lv1（前5章）：势利同事/小混混/前女友现男友
Lv2（5-20章）：富二代/小老板/势利亲戚中的"能人"
Lv3（20-50章）：二线家族/行业大佬/地区势力
Lv4（50-100章）：一线家族/省级人物/全国知名人物
Lv5（100章+）：顶级豪门/国家级人物/国际势力
```

### 节奏建议

- **每章至少一个小爽点**（一次被人轻视后的小反转、一次能力展示）
- **每3-5章一次完整打脸**（从被轻视到反转到震惊的完整弧线）
- **每10章一次身份/实力升级**（进入新圈子/新层级）
- **感情线不能断**：每5-8章要有感情推进的内容
- **不要连续3章以上都是打斗/冲突**，要有日常生活的调剂

## 6. 金手指/系统设计建议

### 常见金手指类型

- **异能类**：透视眼、预知能力、超强体质、念力
- **传承类**：神医传承、武道传承、鉴宝传承（最好有"师父"的灵魂/手册指导）
- **身份类**：隐藏的顶级家族继承人、退役特种兵王、国际组织成员
- **系统类**：神豪系统（每天打钱）、直播系统、鉴宝系统
- **混合类**：身份+能力组合（退伍兵王+医术传承，家族少爷+异能觉醒）

### 都市金手指设计要点

1. **要有合理性包装**：都市文贴近现实，金手指不能太离谱
   - 好的包装：前世记忆、神秘传承、基因变异、家族秘密
   - 差的包装：突然从天上掉下一个系统（太生硬）
2. **能力展示要有场景**：透视眼 → 赌石/鉴宝/看病；武功 → 保镖/格斗/黑市
3. **不能一开始就满级**：能力要有成长空间（初级透视只能看5米，后期能看穿墙壁）
4. **能力要能赚钱**：都市文读者很看重主角的财富增长
5. **要有暴露的风险**：能力被发现可能被研究/被觊觎/被组织盯上

## 7. 番茄平台注意事项

### 展示规范
- 场景描写要具体但不啰嗦（"五星级酒店的总统套房""市中心CBD顶楼办公室"，一句话带过）
- 金钱数字要具体化（"账户余额从2000变成2000万"比"赚了很多钱"好十倍）
- 打脸对象的"变脸"反应要写充分（脸色变化、内心独白、态度反转——这是读者最爱看的部分）
- 大佬出场要有排面（车队/随从/周围人的反应烘托）

### 数据建议
- 都市文在番茄竞争极激烈，开局必须快和爽
- 前3章必须完成"被看不起→展示实力→第一波打脸"的完整弧线
- 每章字数建议2000-2500字
- 章末钩子用"更大的人物出场""更大的危机来临""女主有难"效果最好
- 标题要直接暴力："震惊！""全场跪下""你叫我什么？"类标题点击率高

### 合规注意
- **不要使用真实地名、公司名、品牌名**（用虚构名称替代）
- 不要有太露骨的感情描写（暗示到位即可，不要明写）
- 不要有官场/政治斗争的具体描写（可以模糊提及"上面有人"，但不要具体到哪个部门/职位）
- 打斗场面不要涉及警察/军队的负面描写
- 金融操作不要太具体（不要写成炒股教程）
- 不要出现赌博教学性质的内容

## 8. 避免的陷阱

### 常见写崩原因

1. **打脸太重复**：每次都是同一个模式（被看不起→亮身份→震惊） → 打脸方式要多样化（技术碾压/财富碾压/身份碾压/人脉碾压交替使用）
2. **主角太装**：明明可以直接解决却非要装逼 → 装逼要有理由（低调保护自己/不想暴露身份/给对方面子）
3. **配角脸谱化**：所有看不起主角的人都一个样 → 给反派不同的动机和性格
4. **感情线太后宫**：每个女性角色都喜欢主角 → 番茄对后宫内容管控严格，感情线要收着写
5. **金手指暴露无后果**：主角到处展示超能力却没人管 → 要有组织/势力来调查/拉拢
6. **财富增长不合理**：一个月从穷光蛋变成亿万富翁 → 财富增长要有合理路径（投资→创业→并购）
7. **社会背景失真**：对豪门/上流社会的描写太夸张或太离谱 → 可以参考现实做夸张化处理，但基本逻辑要通
8. **反派层级断档**：从小混混直接跳到国际大佬 → 反派层级要连续递进
9. **日常生活缺失**：全程打打杀杀没有日常 → 适当的日常（吃饭、聊天、逛街）调节节奏

### 自检清单

写作时定期检查：
- [ ] 最近5章是否有打脸爽点？
- [ ] 主角的财富/地位是否在稳步提升？
- [ ] 感情线是否在推进（不能断太久）？
- [ ] 打脸方式是否足够多样化？
- [ ] 是否有"新的更强对手"出现来推动剧情？
- [ ] 主角的金手指是否有暴露风险在积累？
