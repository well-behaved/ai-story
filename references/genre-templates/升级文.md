# 升级文写作模板

> **加载时机**：当 story-start 流程中用户选择"升级文""修仙文""玄幻文"题材，或 AI 判断故事构思属于打怪升级变强类型时，加载本模板作为写作指导。

---

## 1. 题材定义与核心卖点

升级文（含修仙文、玄幻文）是网文最经典、读者群体最大的题材。核心是主角在一个有明确力量等级的世界中，通过修炼、战斗、奇遇不断提升实力，从弱者成长为强者。

**读者为什么爱看：**
- **明确的成长感**：每一次突破都有清晰的等级标识，读者能感受到主角"确实变强了"
- **越级战斗的爽感**：以弱胜强是最大的high点，永远不会看腻
- **探索未知的好奇心**：更高的等级意味着更广阔的世界，一山更比一山高
- **竞技对抗的刺激**：同辈之间的争锋、大比武、排名赛，天然的看点
- **奇遇的期待**：秘境、洞府、传承，总有未知的宝藏在前方等待

**数据表现**：升级文是全品类中体量最大的，在番茄拥有最广泛的读者基础。长篇优势明显，能写到数百万字。新书期数据表现稳定，读者粘性高。

## 2. 典型世界观框架

### 力量体系（九级制，最为通用）

| 等级 | 名称（可自定义） | 能力特征 | 社会地位 |
|------|-----------------|----------|----------|
| 1级 | 淬体/炼体 | 超越常人体魄 | 普通武者 |
| 2级 | 聚气/练气 | 调动天地灵气 | 小有名气 |
| 3级 | 凝元/筑基 | 形成内丹/气海 | 一方小强者 |
| 4级 | 金丹/灵核 | 质变，寿命延长 | 城主/长老级 |
| 5级 | 元婴/化神 | 神魂出窍，飞行 | 宗门级强者 |
| 6级 | 合体/化圣 | 天地共鸣 | 一域之主 |
| 7级 | 大乘/帝境 | 掌控法则 | 一界至强 |
| 8级 | 渡劫/半神 | 打破界限 | 跨界存在 |
| 9级 | 真仙/神王 | 永恒不灭 | 传说级别 |

**设计要点**：
- 每个大等级内可分3个小阶（初期/中期/巅峰），增加过渡层次
- 每次突破大等级要有明显的质变（不只是数字提升，要有新能力）
- 等级名称要和世界观风格统一（仙侠用修仙术语，西幻用骑士/法师术语）

### 空间结构（由小到大）

```
村庄/城镇 → 郡城/王城 → 宗门/学院 → 王朝/帝国 → 域/界 → 上界/神界
```

每升一个等级区间，舞台就扩大一层。这是升级文"一山更比一山高"的核心结构。

### 资源体系

- **功法**：修炼方法，分等级（凡级/玄级/地级/天级/圣级）
- **灵石/晶石**：通用货币+修炼资源
- **丹药**：辅助修炼/恢复/战斗
- **武器/法宝**：战斗装备，同样分等级
- **阵法/符箓**：辅助手段

## 3. 典型角色配置

### 主角
- **起点设计**（两种经典模式）：
  - **天才陨落型**：曾经的天才因某种原因失去天赋/被废修为，后获得机缘重新崛起（更有戏剧性）
  - **平凡觉醒型**：普通资质，被人看不起，突然获得觉醒/传承/变异（更有代入感）
- **性格**：坚韧不屈，有仇必报但不滥杀，对朋友忠诚，面对强者不卑不亢
- **隐藏优势**：除了主要金手指外，要有1-2个后期才显露的特质（特殊体质、特殊血脉等）

### 天才对手（同辈竞争者）
- 同龄人中的天才，是主角的参照物和追赶目标
- 前期实力压过主角，中期被主角赶上，后期被主角超越
- 可以是亦敌亦友的关系，不要写成纯反派（有深度的对手更好看）
- 每个新舞台都要有一个"当前阶段的天才"

### 红颜/伙伴
- 与主角并肩战斗，有自己的成长线
- 不要沦为花瓶，要有独立的能力和性格
- 和主角的关系递进：相遇→磨合→信任→并肩→超越普通关系
- 可以有实力差，但不能差太多（否则后期拖累剧情）

### 导师/前辈
- 经典的"老爷爷"角色（戒指里的灵魂/隐居老人/封印的强者）
- 作用：传授功法、解答疑惑、在关键时刻指点（但不能直接出手解决问题）
- 要有退出机制：到一定阶段后导师隐退/消失/牺牲，让主角独立面对

### 各阶段反派
- **小反派**（5-15章出现）：本地恶霸、同门欺凌者
- **中反派**（20-50章）：宗门内权势人物、对立势力年轻一代
- **大反派**（50-100章）：幕后黑手、域级强者、主角的宿命之敌

## 4. 典型情节结构

### 开局（第1-5章）：激励事件

**模式A：天才陨落**
```
第1章：展示主角曾经的辉煌+现在的落魄（被退婚/被驱逐/被嘲讽）
第2章：获得机缘（吞噬丹药/神秘传承/穿越者灵魂）
第3章：重新修炼，发现自己进步神速
第4章：第一次战斗，以低等级击败高等级对手
第5章：引起关注/被人注意，新的挑战出现
```

**模式B：平凡觉醒**
```
第1章：普通人的挣扎（资质测试垫底/被嘲笑/差点被淘汰）
第2章：意外获得觉醒（秘境奇遇/祖传之物激活/濒死突破）
第3章：发现自己有了超越常人的修炼速度
第4章：第一次实战，证明自己不是废物
第5章：进入更大舞台（被宗门录取/获得参赛资格）
```

### 发展（第6-100章）：升级循环

**标准升级弧线（每15-25章一个循环）：**
```
进入新舞台 → 被新环境的强者压制 → 获得修炼资源/功法 → 闭关/特训
→ 突破等级 → 用新实力打败之前压制自己的人 → 发现更大的世界/更强的对手 → 循环
```

**穿插副本弧线（每20-30章一个）：**
```
发现秘境/遗迹信息 → 准备进入（组队/竞争名额） → 探索秘境（机关/怪物/争夺）
→ 获得核心传承/宝物 → 秘境结束，实力大增 → 回到主线面对新挑战
```

### 大比武/竞赛弧线（每50-80章一个）

```
大比武消息传来 → 备战修炼（提升+学新技能） → 预赛（展示基本实力）
→ 正赛（逐轮对战，难度递增） → 遇到天才对手（苦战） → 决赛/半决赛突破等级
→ 最终胜利（或惜败但赢得尊重） → 获得大赛奖励+名声 → 更大舞台的邀请
```

### 高潮（每80-120章一个大转折）

```
大危机降临（灭宗/灭城/强者追杀） → 主角在绝境中爆发
→ 突破关键等级/觉醒隐藏能力 → 逆天改命 → 旧世界格局打破 → 新世界展开
```

## 5. 典型爽点类型和节奏

### 核心爽点清单

| 爽点类型 | 频率 | 说明 |
|----------|------|------|
| 越级战斗 | 每5-8章 | 以弱胜强，核心爽点 |
| 等级突破 | 每15-25章 | 小等级突破频繁，大等级突破是大事件 |
| 打脸嘲讽者 | 每8-12章 | "他怎么可能？！"类反应 |
| 获得传承/奇遇 | 每20-30章 | 秘境收获、偶遇仙缘 |
| 炼丹/炼器成功 | 每15-20章 | 副职业带来的成就感 |
| 身份揭露/震惊 | 每30-50章 | 隐藏身份被发现，众人震惊 |
| 大比武获胜 | 每50-80章 | 公开场合证明实力 |

### 战斗节奏

- 单场战斗控制在 **500-800字**（小战斗300-500字，Boss战可到1000-1500字）
- 不要逐招描写，重点写转折点和致命一击
- 等级差异要视觉化：低等级看高等级的攻击"根本看不清"，碾压要有"一招秒杀"的体现
- 战斗中穿插旁观者的惊叹（"这怎么可能！""他居然能接住这一招？！"）

### 升级节奏

- **前期**（前50章）：10-20章一个大等级，快速成长制造爽感
- **中期**（50-200章）：30-50章一个大等级，加入更多剧情和人物
- **后期**（200章+）：80-120章一个大等级，每次突破都是大事件

## 6. 金手指/系统设计建议

### 常见金手指类型

- **特殊体质**：天生或后天获得的特殊体质，修炼速度加倍/特殊能力
- **神秘空间**：随身携带的独立空间（内有灵泉/药田/时间加速）
- **吞噬能力**：可以吞噬他人功力/妖兽精华来提升自己
- **前辈传承**：强者灵魂教导+遗留宝物/功法
- **先知/天眼**：能看到机缘所在/危险预警
- **炼丹/炼器天赋**：副职业碾压，用丹药换资源

### 金手指设计原则

1. **不能太强**：金手指是加速器而非无敌挂，主角仍需努力
2. **要有成长性**：金手指本身也能升级，和主角一起变强
3. **要有限制**：使用次数/冷却时间/副作用/消耗资源
4. **要能制造剧情**：金手指暴露的风险、被人觊觎、不可控的副作用
5. **只给一个核心金手指**：辅助的可以有，但核心金手指只需一个

## 7. 番茄平台注意事项

### 展示规范
- 等级突破要有仪式感（天象异变/灵气漩涡/光柱冲天）
- 等级名称在文中要统一，不要混用
- 战斗场面不要太长，番茄读者偏好快节奏
- 数据要具体化："一拳轰出，三丈内的地面全部碎裂"比"一拳很强"好

### 数据建议
- 升级文在番茄属于大品类，读者基数大但竞争也大
- 开局5章要完成"被低估→获得机缘→第一次证明自己"的完整弧线
- 每章字数建议2000-2500字
- 章末钩子用"即将突破""强敌出现""秘境开启"效果最好

### 合规注意
- 战斗描写不要过于血腥（"鲜血四溅"可以，但不要详细描写伤口）
- 不要有邪教/血祭等敏感描写
- 宗门体系不要影射现实组织
- 杀戮要有正当理由，避免主角无差别杀人

## 8. 避免的陷阱

### 常见写崩原因

1. **力量体系崩坏**：前面说金丹期能毁山，后面又说筑基也能 → 每个等级的能力上限要提前规划好并记录
2. **升级太随意**：想升就升，没有积累过程 → 每次升级要有铺垫（修炼+机缘+感悟）
3. **反派智商下线**：反派永远送经验 → 给反派合理的行为逻辑和还手能力
4. **配角工具化**：朋友只在需要时出现 → 给重要配角独立的故事线
5. **地图炮**：每到一个新地方就一堆人无缘无故找主角麻烦 → 冲突要有合理的利益驱动
6. **功法设定混乱**：功法太多记不住 → 核心功法1-2个，辅助技能3-5个足矣
7. **闭关写太多**：整章就在写修炼突破 → 闭关过程简写，重点写突破后的变化和应用
8. **世界观膨胀太快**：刚写30章就出了三个大陆五个位面 → 一个阶段聚焦一个舞台
9. **女性角色工具化**：所有女角色都是花瓶或奖品 → 番茄对这类内容审核趋严

### 自检清单

写作时定期检查：
- [ ] 当前等级体系是否前后一致？
- [ ] 主角的战斗力提升是否有合理的铺垫？
- [ ] 最近10章内是否有越级战斗或等级突破的爽点？
- [ ] 当前舞台的角色是否都有存在意义？
- [ ] 是否有"一山更比一山高"的感觉？
