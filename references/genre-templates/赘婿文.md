# 赘婿文写作模板

> **加载时机**：当 story-start 流程中用户选择"赘婿文""上门女婿"题材，或 AI 判断故事构思包含"入赘""被岳家看不起""隐藏身份逆袭"等赘婿元素时，加载本模板作为写作指导。

---

## 1. 题材定义与核心卖点

赘婿文是番茄小说平台上的超级爆款品类。核心设定是：主角作为上门女婿入赘到妻子家族，被岳家全员瞧不起、遭受各种屈辱，但主角实际拥有隐藏的顶级身份或绝世能力，通过一次次事件逐步展露实力，最终让所有轻视他的人目瞪口呆。

**读者为什么爱看：**
- **"先抑后扬"的极致体验**：开局越憋屈，后面越解气。赘婿文把这个模式推向了极端
- **社会隐痛的代入**：很多男性读者在现实中有过被人看不起/被势利对待的经历，赘婿文直击这个痛点
- **"打脸"频率最高**：几乎每3-5章就有一波打脸，读者的爽感持续不断
- **身份反转的巨大冲击**：从"最没用的女婿"到"所有人跪下"，反差越大越爽
- **层层揭秘的悬念**：主角到底是什么身份？为什么甘愿做赘婿？这个悬念能撑很长的篇幅

**数据表现**：赘婿文在番茄的数据表现极其亮眼，追读率和完读率都在高位。开篇几章的留存率特别高（"受气→打脸"的模式天然勾人），付费转化好。但竞争非常激烈，要找差异化。

## 2. 典型世界观框架

### 社会层级结构

赘婿文的世界观是"阶层金字塔"：

```
第一层：岳家（中产/小富之家，在本市有一定地位但不是顶流）
第二层：本市真正的豪门/权贵（岳家巴结的对象）
第三层：省级/区域级大家族（本市豪门也要仰望的存在）
第四层：国家级隐藏家族/军方高层/国际势力
第五层：主角真正的背景所在的层级（通常是第三层或第四层）
```

### 主角入赘的原因设定

必须给主角一个合理的"暂时隐忍"的理由：
- **使命型**：被家族派来保护妻子/执行秘密任务/寻找某样东西
- **承诺型**：对已故长辈的承诺（"照顾好他家的女儿"）
- **修养型**：重伤/中毒需要时间恢复，暂时隐藏身份
- **考验型**：家族对继承人的考验，需要从底层证明自己
- **情感型**：真的喜欢妻子，甘愿为她忍受（但这个理由要搭配其他理由才够有力）

### 岳家的家族结构

标准配置：
- **岳父**：有一定能力但胆小怕事/要面子，对主角态度复杂（有时同情但不敢说话）
- **岳母**：赘婿文中最"势利"的角色，几乎每章都在嫌弃主角，打脸后变脸最快
- **妻子**：初期冷漠/无视/被家族影响而轻视主角，但不能太过分（后期要洗白，太过分了洗不白）
- **小舅子/大姨子等亲戚**：嘲讽主角的主力，充当"打脸靶子"
- **岳家的事业**：一个中等公司/生意，后续会遇到经营危机（给主角救场的机会）

## 3. 典型角色配置

### 主角（隐藏身份的赘婿）

**表面**：看起来没用的上门女婿，做饭洗衣接孩子（如果有孩子的话），被岳家当废物
**内在**：隐藏的顶级存在——

常见隐藏身份（选一个主身份 + 1-2个副身份）：
| 隐藏身份 | 特点 | 揭露方式 |
|----------|------|----------|
| 豪门继承人 | 财富碾压 | 大佬称呼"少主"/黑卡亮出 |
| 退伍兵王 | 武力碾压 | 一招制敌/老战友敬礼 |
| 神医传人 | 医术碾压 | 妙手回春/名医跪求指教 |
| 商业巨鳄 | 商业碾压 | 一个电话让对方公司倒闭 |
| 古武传人 | 身份碾压 | 古武世家晚辈见面叫"前辈" |

**性格**：隐忍但不窝囊（有底线，不是任人欺负的沙包），平时懒得计较，触碰底线时果断出手，对妻子有真感情。

### 妻子（最关键的角色之一）

**核心设计原则：妻子不能太恶毒。**

妻子是后期要和主角走到一起的人，如果前期太过分（当众羞辱/动手打/直接要离婚），读者后期无法接受她的洗白。

**合理的妻子态度梯度：**
- 冷漠忽视（可以） → "你自己吃饭吧，我不等你了"
- 不信任能力（可以） → "这种事你就别掺和了"
- 被家族裹挟的轻视（可以） → 在家族聚会上不为主角说话
- 直接侮辱（不行） → 这样后面很难洗白
- 动手打人（绝对不行） → 读者会直接弃书

**妻子的转变弧线**：冷漠→好奇（主角怎么变了？）→ 重新审视→心疼→明确心意

### 势利岳母
- 赘婿文的"标准反派"，几乎每个赘婿文都有
- 每次家庭聚会都要嫌弃主角、说风凉话、拿主角和别人家女婿比
- 打脸后的反应是读者最爱看的：目瞪口呆→不敢相信→讨好巴结
- 但也不能太丑化：后期可以给一个"为了女儿好才这样"的解释

### 嚣张姐夫/小舅子/堂弟
- 打脸的主要靶子之一
- 通常自己有点小钱小势，瞧不起赘婿
- 经常在家庭聚会上挑衅、炫耀、对比
- 他们的"后台"也是层级递进的打脸对象

### 外部大佬（身份揭露的关键人物）
- 主角真正圈子里的人物（老战友/家族管事/商业伙伴）
- 在公开场合遇到主角时的反应："X少/X爷/老大，您怎么在这？"
- 这个人的出场永远是最大的爽点——全场所有看不起主角的人集体石化

## 4. 典型情节结构

### 开局（第1-5章）：建立憋屈感+第一波反转

**关键原则：受气时长控制在3-5章，不能更多。读者忍耐力有限。**

```
第1章：展示主角作为赘婿被轻视的日常（做饭被嫌弃/家庭聚会被嘲讽/被小舅子挤兑）
       暗线：暗示主角有隐藏身份（一个神秘来电/一个收到的快递/一段内心独白）
第2章：憋屈升级——某个外人当众羞辱主角，岳家没人帮忙说话
       妻子也保持沉默或帮着说主角（但程度控制好）
第3章：触发事件——岳家/妻子遇到麻烦（生意危机/被人欺负/健康问题）
       所有人束手无策，甚至之前嘲讽主角的人也帮不了
第4章：主角出手——一个电话/一出手/一句话解决问题
       第一个外部大佬出场，对主角毕恭毕敬
第5章：全家震惊——但还没完全相信，以为是巧合/运气
       更大的事件伏笔埋下
```

### 发展（第6-80章）：层层打脸+身份逐步揭露

**家庭打脸弧线（每5-8章一个）：**
```
家庭聚会/亲戚来访 → 主角被挤兑/嘲讽 → 亲戚的后台/生意出问题
→ 主角轻松解决（或主角的一个电话让对方的后台来道歉）
→ 亲戚目瞪口呆 → 岳母/岳父态度微妙变化
```

**商业打脸弧线（每8-15章一个）：**
```
岳家公司遇到商业危机（大客户被抢/被竞争对手打压/资金链断裂）
→ 岳家所有人想办法无果 → 有人嘲讽"你那赘婿能帮什么忙"
→ 主角暗中出手（调动资源/联系人脉/亲自谈判）
→ 问题解决，对方大佬亲自来道歉 → 岳家逐渐意识到主角不简单
```

**身份揭露弧线（每15-25章一个，递进式）：**
```
第一次：某个有身份的人对主角恭敬 → 岳家以为认错人了
第二次：更大的人物对主角恭敬 → 岳家开始怀疑
第三次：主角的真实资源曝光一角 → 岳家震惊但不知全貌
第四次：主角的核心身份揭露 → 全家跪了/全城震动
```

**感情线弧线（贯穿全文）：**
```
妻子冷漠 → 主角为岳家解决问题，妻子开始重新审视
→ 日常相处中发现主角的好（做饭/照顾/默默付出）
→ 知道主角的部分真实身份后心疼（原来他一直在忍）
→ 明确自己的感情 → 两人真正走到一起
```

### 高潮（每50-80章一个大高潮）

**大型身份揭露型高潮：**
```
大型公开场合（商业宴会/家族大聚会/行业峰会）
→ 主角被当众最严重的一次羞辱（"不过是个赘婿"）
→ 最高级别的大佬登场 → 当众揭示主角身份
→ "他就是XX家族的唯一继承人/龙组最强战士/XX集团真正的幕后老板"
→ 全场跪/全场静默/所有曾经嘲讽的人脸色惨白
→ 妻子流泪/岳母颤抖/势利亲戚吓瘫
```

## 5. 典型爽点类型和节奏

### 核心爽点清单

| 爽点类型 | 频率 | 说明 |
|----------|------|------|
| 家庭打脸 | 每3-5章 | 势利亲戚被教训，赘婿文核心爽点 |
| 大佬敬礼 | 每8-12章 | 外部大人物对主角毕恭毕敬 |
| 身份局部揭露 | 每15-20章 | 主角身份冰山一角被发现 |
| 救岳家于水火 | 每10-15章 | 岳家遇难，主角出手化解 |
| 妻子态度转变 | 每8-12章 | 妻子的一个温柔举动/一句维护 |
| 情敌被碾压 | 每10-15章 | 追求妻子的"优秀男人"被主角秒杀 |
| 完整身份揭露 | 每50-80章 | 大范围公开身份，震撼全场 |

### "憋屈-爽"的节奏控制

赘婿文的核心节奏是"先憋后爽"，但要控制比例：

- **憋屈内容占比不超过30%**：每个弧线中，受气的部分1-2章，打脸的部分2-3章
- **不要连续受气超过3章**：读者忍耐力有限，超过3章没爽点会流失
- **打脸后给一小段"甜"**：打完脸后安排一段妻子的态度变化/家人的认可，给读者喘息
- **每个打脸弧线的爽度要递增**：不能每次都是同样程度的打脸

### 打脸对象层级表

```
Lv1（第1-10章）：势利亲戚、小舅子、嘲讽主角的邻居
Lv2（第10-30章）：亲戚的后台、追求妻子的富二代、岳家的商业对手
Lv3（第30-60章）：本市豪门、行业大佬、看不起岳家的势力
Lv4（第60-100章）：省级家族、更大的势力、主角过去的"敌人"
Lv5（第100章+）：国家级势力、主角身份背后的终极对手
```

## 6. 金手指/系统设计建议

### 赘婿文的"金手指"就是隐藏身份

赘婿文不太需要传统意义上的金手指（系统/异能），主角的隐藏身份和背后的资源网络就是最大的金手指。

### 隐藏身份设计原则

1. **身份要有"天花板感"**：至少是本市最顶级的存在之一，否则打脸不够爽
2. **身份要有层级**：不要一次揭露完毕，要一层一层剥洋葱
   - 第一层：有钱（解决小问题用钱）
   - 第二层：有人脉（一个电话搞定中等问题）
   - 第三层：有权势（动用组织力量解决大问题）
   - 第四层：有身份（真实身份让所有人震惊）
3. **身份要有"不能说的理由"**：如果没有限制，主角直接亮身份就完事了，就没故事了
   - 家族规矩：继承人必须在外历练，不能动用家族资源
   - 使命限制：在任务完成前不能暴露身份
   - 保护目的：暴露身份会给妻子带来危险
   - 个人选择：想看看妻子是否真心爱自己而非身份

### 资源网络设计

主角可调动的资源要分层，随剧情逐步解锁：
- **初期可用**：个人武力（能打）+ 少量现金 + 1-2个忠诚下属
- **中期解锁**：大量资金 + 商业资源 + 一支团队
- **后期解锁**：家族全部资源 + 军方/组织关系 + 顶级人脉
- **终极**：真实身份的全部权力

## 7. 番茄平台注意事项

### 展示规范
- 打脸反应要写得详细而有层次（先是不敢相信→然后恐惧→最后讨好/道歉），这是读者最爱看的部分
- 大佬出场要有排面描写（豪车车队/一群保镖/所有人让路）
- 身份揭露时的"称呼"要有冲击力（"X少爷！""龙组第一人！""恩师在上！"）
- 金钱数字要具体化，读者喜欢看到具体的数字对比

### 数据建议
- 赘婿文在番茄数据非常好，但竞争极其激烈（同类型书太多）
- **差异化是关键**：不能写得和别人一模一样，要在隐藏身份/妻子性格/打脸方式上找到独特卖点
- 前3章就必须完成"受气→第一波打脸"的完整弧线
- 每章字数建议2000-2200字，短而爽
- 章末钩子用"更大的人物即将出现""妻子发现了什么""新的挑战来临"效果最好
- 标题要有冲击力，常见有效标题模式："全场跪下""你叫我什么？""她后悔了""真正的身份"

### 合规注意
- 岳家的"虐"不能涉及家暴或犯罪行为（冷嘲热讽可以，动手打人不行）
- 主角的报复手段要合法合规（商业手段/社交碾压，不要违法犯罪）
- 不要有太露骨的感情描写
- 主角的隐藏组织不要涉及敏感机构

## 8. 避免的陷阱

### 常见写崩原因

1. **受气太久**：前10章还在被欺负没有任何反转 → 最多3-5章必须给第一波爽点
2. **妻子太恶毒**：当众扇耳光/说"你不配"等极端表现 → 后期洗白读者不买账，直接弃书
3. **打脸太套路**：每次都是"嘲讽→一个电话→大佬来了→跪" → 打脸方式要多变（技术碾压/财富碾压/身份碾压/武力碾压交替）
4. **身份揭露太快**：20章就全部揭露了 → 身份要像剥洋葱，一层一层来
5. **身份揭露太慢**：100章了还在被欺负 → 揭露节奏要紧凑，每15-20章要有实质进展
6. **隐忍理由不合理**：堂堂豪门少爷/兵王为什么要忍？如果理由不够充分，读者会觉得主角窝囊而非隐忍
7. **反派全是智障**：每个看不起主角的人都是蠢货 → 要有聪明的对手，给主角真正的挑战
8. **配角工具化**：所有亲戚都只负责"嘲讽→被打脸" → 给1-2个亲戚安排转变弧线（从看不起到认可）
9. **感情线断裂**：只顾打脸忘记经营主角和妻子的关系 → 感情线是赘婿文的另一条命脉

### 差异化建议（在番茄突围的关键）

既然赘婿文竞争激烈，以下是可以尝试的差异化方向：
- **独特的隐藏身份**：不要再写"豪门少爷""退伍兵王"了，考虑更新颖的身份（世界级厨神/顶级黑客/考古大师）
- **妻子的独特设定**：不要写"冰山美女总裁"了，考虑给妻子更有趣的职业和性格
- **岳家不是纯反面**：给岳家一些合理的苦衷（负债/被胁迫/有不得已的原因），增加角色深度
- **主角有明确的"赘婿期限"**：比如"三年之约"到期就走，增加时间紧迫感
- **双线叙事**：赘婿日常+暗中的大事件交替，增加故事深度

### 自检清单

写作时定期检查：
- [ ] 最近5章"憋屈"和"爽"的比例是否合理（3:7左右）？
- [ ] 妻子的态度是否在逐步变化（不能一直冷漠也不能突然热情）？
- [ ] 身份揭露是否在按计划递进？
- [ ] 打脸方式是否有变化（不是每次都靠一个电话）？
- [ ] 是否有新的更强对手出现来保持紧张感？
- [ ] 主角"忍耐"是否有合理的理由支撑？
