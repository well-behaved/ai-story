# 去AI味完整规则

> **用途**：写作全流程的核心过滤规则。每次生成正文内容后必须加载此文件进行自检。
> **加载时机**：写作阶段的每次输出前/输出后校验；修改润色阶段全程加载。

## 一、禁用词表

以下词汇/短语在正文中禁止使用。出现即删除或替换为具体描写。

### 1. 转折/连接类（禁用）

| 序号 | 禁用词 | 替代策略 |
|------|--------|----------|
| 1 | 然而 | 直接写转折后的事实，靠语义对比产生转折感 |
| 2 | 此外 | 删除，另起一段直接写 |
| 3 | 值得注意的是 | 删除，直接陈述事实 |
| 4 | 总的来说 | 删除，不做总结 |
| 5 | 显然 | 删除，让读者自己判断 |
| 6 | 不难看出 | 删除 |
| 7 | 不言而喻 | 删除 |
| 8 | 毋庸置疑 | 删除 |
| 9 | 综上所述 | 删除，小说不需要总结 |
| 10 | 事实上 | 删除或改为具体细节 |
| 11 | 由此可见 | 删除 |
| 12 | 与此同时 | 改用时间标记或直接切场景 |
| 13 | 换言之 | 删除，不要解释性复述 |
| 14 | 不仅如此 | 删除，直接写递进内容 |
| 15 | 从某种意义上说 | 删除 |
| 16 | 首先...其次...最后 | 删除连接词，用场景/动作自然推进 |
| 17 | 一方面...另一方面 | 分开写，不标注 |
| 18 | 正因如此 | 删除 |

## 二、禁用句式模式

以下句式结构出现即触发修改。

### 1. 三连排比
- **模式**："他看到了X，看到了Y，看到了Z"
- **问题**：AI最爱用的结构，一眼假
- **修改**：只保留最有冲击力的一个，其余删除或改为不同句式
- **反例**：他看到了倒塌的城墙，看到了四散的百姓，看到了遮天的烟尘。
- **正例**：城墙塌了半边。碎石下面压着人，还在动。

### 2. "不是X，而是Y"分析模式
- **模式**："他害怕的不是死亡，而是孤独"
- **问题**：分析腔，像在写论文
- **修改**：通过场景/动作展示，而非直接分析
- **反例**：他恐惧的不是面前的敌人，而是失去同伴的可能。
- **正例**：他挡在队友前面的时候没犹豫。

### 3. "如果说X，那么Y就是Z"总结模式
- **模式**：总结概括型句式
- **问题**：说教感，跳出故事
- **修改**：删除，让故事自己说话

### 4. 连续句首重复
- **模式**：连续3+句以相同词开头（"他...他...他..."）
- **问题**：AI缺乏变化能力的表现
- **修改**：变换主语、省略主语、用动作开头、用环境开头

### 5. 重复定语结构
- **模式**："一个XX的人，一个XX的人"
- **问题**：模板化描述
- **修改**：用不同句式分别描写，突出差异

### 6. 解释模式
- **模式**："他知道，这一切都是因为..."
- **问题**：告知而非展示
- **修改**：删除解释，让读者从行动中推测因果

### 7. 初体验模式
- **模式**："XX，这是他第一次感受到XX"
- **问题**：直接点题，破坏留白
- **修改**：只写感受的外化，不标注"第一次"

### 8. 段末升华
- **模式**："这一刻，他终于..."/"从此以后..."
- **问题**：AI标志性的段落收束方式
- **修改**：在高点切断，或接一个新动作/新信息

### 9. 过度比喻
- **模式**："仿佛...一般"/"如同...一样"
- **问题**：比喻过密、比喻套路
- **修改**：每千字最多1个比喻，且必须贴合角色认知（农民用庄稼比喻，剑客用刀剑比喻）

### 10. 平衡套话
- **模式**："虽然...但是..."的辩证式心理描写
- **问题**：圆滑、不痛不痒
- **修改**：只写一面——角色当下最强烈的那个情绪

### 11. 结尾反问
- **模式**："难道不是吗？"/"这难道就是...？"
- **问题**：做作
- **修改**：改为陈述或直接推进

### 12. 百感交集模式
- **模式**："看着眼前的一切，XX心中百感交集"
- **问题**：空洞，什么都没说
- **修改**：选一种最主要的情感，用一个微小动作表达

### 13. 动作+心理双写
- **模式**："他握紧了拳头，心中暗暗发誓"
- **问题**：重复表达同一信息
- **修改**：只保留一个——要么动作，要么内心独白

### 14. 旁白解说
- **模式**："殊不知，这只是个开始"/"如果他知道接下来会发生什么..."
- **问题**：打破叙事视角，AI常用的廉价悬念
- **修改**：删除，让事件自然发展

### 15. "的"字堆叠
- **模式**：连续3个以上"的"字定语修饰一个名词
- **问题**：臃肿、节奏拖沓
- **反例**：那个穿着一身破旧的蓝色的打满补丁的衣服的老人
- **正例**：老人身上的蓝衣全是补丁，洗得快要看不出颜色。

### 16. 水字数模式
- **模式**："时间一分一秒地过去"/"沉默在两人之间蔓延"
- **问题**：空转，无新信息
- **修改**：要么跳过时间，要么在等待中发生微小事件

### 17. 哲理结尾
- **模式**："或许，这就是命运的安排"/"人生就是如此"
- **问题**：说教、鸡汤
- **修改**：结尾落在具体事物上——一个声音、一个动作、一个画面

### 18. 程式化笑容
- **模式**："XX笑了，笑得很灿烂/苦涩/释然"
- **问题**：告诉而不是展示
- **修改**：写笑容具体是什么样（露出牙/眼角皱纹/笑出声/无声地笑）

### 19. 气息模式
- **模式**："空气中弥漫着XX的气息"
- **问题**：廉价的氛围描写
- **修改**：用具体的感官细节替代（具体的气味、温度变化、声音）

### 20. 回荡模式
- **模式**："XX的声音在XX中回荡"
- **问题**：套路化的场景收束
- **修改**：写声音消失后的安静中发生了什么

### 21. 千篇一律的场景开头
- **模式**："阳光透过窗户洒在..."/"月光如水..."
- **问题**：AI默认的场景开启方式
- **修改**：从动作/对话/声音开头

### 22. 列举式信息倾倒
- **模式**：连续5+句介绍背景/设定/人物信息
- **问题**：读者会跳过
- **修改**：将信息融入冲突和对话中

## 三、六大门禁详细规则

### 门禁1：禁用词过滤

**触发条件**：正文中出现第一节列出的任何禁用词。

**检测方法**：
- 逐段扫描，匹配禁用词表
- 统计"仿佛/好似"等限频词的出现密度
- 标记所有匹配项

**修改策略**：
- 直接删除（连接词、升华词）
- 替换为具体描写（动作词、情感词）
- 限频使用（竟然/居然每千字1次，仿佛类每千字1次）

**正面示例**：
```
原文（AI味）：他缓缓走向前方，心中不禁涌起一股莫名的感动。然而，他知道这一切都来之不易。
修改后：他往前走。脚底下全是碎玻璃，每踩一步都嘎吱响。走到头的时候，鼻子酸了一下。
```

```
原文（AI味）：此外，值得注意的是，她的实力远超常人。显然，这并非偶然。
修改后：她一拳砸碎了演练场的石柱。周围练功的弟子全停了手。
```

### 门禁2：句式打散

**触发条件**：连续3句以上使用相同句式结构（主谓宾均齐/长度相近/句首相同）。

**检测方法**：
- 分析连续句的句首词是否重复
- 检查句长是否过于均匀（连续5句以上长度差异<5字）
- 识别排比/对仗结构

**修改策略**：
- 倒装："三天没吃东西了"→"三天了，粒米没沾。"
- 省略主语："他拔出剑。他看向敌人。"→"拔剑。对面那人也动了。"
- 碎片句插入：在完整句之间插入3-5字的短句打破节奏
- 问句替代：将陈述改为角色的自问

**正面示例**：
```
原文（AI味）：
他看着面前的敌人。他握紧了手中的剑。他知道这将是一场恶战。他深吸一口气，准备迎接挑战。

修改后：
对面那人提着刀，刀上还在滴血——不知道是谁的。他攥了攥剑柄。手心全是汗。打吧。
```

```
原文（AI味）：
这座城市很大。街道很宽。人流很密。建筑很高。空气中弥漫着繁华的气息。

修改后：
城挺大。走了半个时辰还没到头，两边全是铺子，卖什么的都有。有一栋楼修了七层高，挡住了半边天。
```

### 门禁3：心理外化

**触发条件**：出现"他/她 + 感到/觉得/想到/意识到 + 情绪/判断词"的直接心理描写。

**检测方法**：
- 匹配模式：主语 + 心理动词 + 情绪名词
- 识别"心中/内心/脑海中"等内部视角标记词

**修改策略**：
- 动作映射：情绪→该情绪下人会做的无意识动作
- 环境映射：情绪→角色注意到的环境细节（恐惧时注意到黑暗/声音）
- 生理反应：情绪→身体不受控的反应
- 对话暗示：情绪→说话方式/内容的变化

**情绪→外化映射表**：

| 情绪 | 动作映射 | 生理反应 | 环境映射 | 对话表现 |
|------|----------|----------|----------|----------|
| 愤怒 | 握拳/咬牙/摔东西/步伐加重 | 太阳穴跳/脸热/呼吸粗 | 注意到碍事的物体 | 声音低/字少/咬字重 |
| 恐惧 | 后退/缩肩/抓住身边物体 | 手抖/冷汗/腿软/胃缩紧 | 注意到阴影/声音/退路 | 声音轻/说不完整/结巴 |
| 悲伤 | 低头/动作变慢/发呆 | 喉咙堵/鼻酸/眼胀 | 注意到安静/缺失物 | 话变少/走神/答非所问 |
| 喜悦 | 动作加快/坐不住/反复确认 | 心跳快/手热/嘴角控不住 | 注意到亮色/好的细节 | 话变多/声音高/重复 |
| 紧张 | 反复摸某物/调整衣服/走来走去 | 口干/手心汗/肌肉紧 | 注意到时间/出口/别人的表情 | 说话快/反问多/犹豫 |
| 愧疚 | 回避目光/缩手/身体后倾 | 胃不舒服/坐立不安 | 注意到对方的表情细节 | 主动换话题/过度解释 |
| 绝望 | 停止动作/放松肌肉/摊坐 | 全身发冷/空白感 | 注意到远处/虚无的东西 | 不说话/说无意义的话 |
| 警觉 | 身体前倾/放轻脚步/手摸武器 | 瞳孔放大/心跳快但稳 | 扫描四周/注意到异常细节 | 压低声音/用短句/指令式 |
| 期待 | 身体前倾/频繁看向目标方向 | 心跳加速/手指动 | 注意到与期待对象相关的事物 | 铺垫性的话/故意不提 |
| 厌恶 | 身体后仰/偏开脸/皱鼻 | 胃部不适/想吐的感觉 | 注意到脏/乱/不协调 | 话变损/用鼻音/词变短 |

**正面示例**：
```
原文（AI味）：他感到一阵恐惧涌上心头。
修改后：他退了半步。后背碰到冰冷的墙壁，衣服瞬间贴在皮肤上——全是汗。

原文（AI味）：她心中涌起一股愧疚感。
修改后：她把筷子放下了。"我吃饱了。"饭还剩大半碗。对面的人什么也没说。

原文（AI味）：他意识到自己犯了一个大错。
修改后：手机屏幕还亮着，发出去的那条消息下面显示"已读"。他盯着那两个字看了十秒钟，然后把手机翻了过去。
```

### 门禁4：节奏打碎

**触发条件**：连续段落的句长过于均匀——中等句长（15-25字）占比超过60%。

**检测方法**：
- 统计每段所有句子的字数
- 计算标准差——标准差<5说明句长太均匀
- 检查是否存在超短句（3-8字）和长句（30+字）的穿插

**修改策略**：
- 在动作紧密处用超短句：3-8字，断裂感，推进快
- 在描写/铺垫处用长句：30+字，信息量大，节奏放缓
- 遵循"短-短-短-长-短"或"长-短-短"的交错模式
- 关键时刻用独立成段的一句话

**正面示例**：
```
原文（AI味，全是中等句）：
他快速穿过走廊来到大厅。大厅里已经聚满了人。每个人脸上都带着紧张的表情。他在人群中找到了自己的位置。然后安静地等待着开始。

修改后（节奏打碎）：
他跑过来的。
走廊那头的门还在晃，大厅里挤满了人，黑压压一片，低着头的、踮着脚的、扒着前面人肩膀往里看的——没一个坐下来。
他挤进去。找到自己的号。
站好。等。
```

```
原文（AI味）：
她握紧了手中的信件。信件上面的字迹模糊不清。她努力辨认了很长时间。终于看清了上面写的内容。那一瞬间她愣住了。

修改后：
信纸上有水渍，墨洇开了一大片。她凑到窗边，侧着脑袋，对着光一个字一个字辨认。"二月初三"——看清了这四个字之后她没再往下看。手垂下去了。信纸落在地上，正面朝下。
```

### 门禁5：对话去腔调

**触发条件**：对话中出现以下问题——
- 过于礼貌（每句带"呢/啊/吧"等语气词）
- 完整句式（现实中人说话不会主谓宾俱全）
- 信息量为零的回应（"嗯，我知道了"、"好的，我明白"）
- 所有人说话语气一样
- 对话承担解说功能

**检测方法**：
- 扫描对话中的语气词密度
- 检查对话句子是否过于完整（超过20字的完整句比例）
- 识别纯回应性质的对话（不推进信息的）
- 检查不同角色的用词是否有区分度

**修改策略**：
- 打断：一个人话没说完被打断/自己打断自己
- 省略：省略主语/宾语/明显的信息（"在哪？""三楼。"）
- 动作替代：用动作代替肯定回答（问"去不去"——对方直接起身）
- 潜台词：说的和想的不一样，靠动作暴露真意
- 口癖区分：每个角色有独特的用词习惯/说话节奏

**正面示例**：
```
原文（AI味）：
"你好，请问你是李明先生吗？"她礼貌地问道。
"是的，我就是李明。请问你有什么事情吗？"他微笑着回答。
"我想和你谈谈关于那个项目的事情，不知道你现在方便吗？"

修改后：
"李明？"
他点了下头。
"那个项目——"她看了眼旁边经过的人，把声音压低了，"能借一步说话吗？"
```

```
原文（AI味）：
"师父，我已经按照您的指示完成了修炼。我感觉自己的实力有了明显的提升。"
"嗯，不错。你的努力值得肯定。继续保持这种状态，你一定能够突破瓶颈的。"

修改后：
"师父。练完了。"
老头连眼皮都没抬，"出拳。"
他出了一拳。
"再来。"
又一拳。
"行了。"老头站起来，走到院子另一头，"后天上擂台，打赢了再来找我。"
```

### 门禁6：结尾去升华

**触发条件**：章节最后3段出现——
- 哲理总结（"人生就是..."/"也许..."）
- 人生感悟（"这一刻他终于明白了..."）
- 命运感叹（"命运的齿轮..."/"一切才刚刚开始"）
- 强行圆满感（"嘴角露出微笑"/"XX心中感到一阵平静"）

**检测方法**：
- 扫描最后3段是否包含升华类禁用词
- 检查最后一句是否是总结/感叹句式
- 判断结尾是否收束过于完整（没留任何未解决的东西）

**修改策略**：
- 悬念替代：结尾指向一个未知（新声音/新人物/意外发现）
- 动作结尾：最后一句是一个具体动作（推门/转身/拿起某物）
- 未完成对话：对话说到一半切断
- 新信息暗示：结尾抛出一个新信息但不展开
- 感官结尾：以一个感官细节收束（一个声音/一种气味/一个画面）

**正面示例**：
```
原文（AI味结尾）：
看着远方的夕阳，他终于明白了师父的话。也许，这就是成长的代价。他微笑着，迈开了脚步。从今以后，他将不再迷茫。

修改后：
天快黑了。他拍了拍裤腿上的土，站起来。
口袋里的传讯符突然热了一下。
他没急着看。先把山顶上的那坛酒喝完了最后一口——冷的。然后他打开传讯符。
上面只有两个字。
```

```
原文（AI味结尾）：
这一战让他明白了，真正的强大不是力量，而是守护之心。他凝望着同伴们，心中涌起一股暖流。这一刻，他终于成长了。

修改后：
他数了数。七个人上去，三个人回来。
"走吧。"
"那他们——"
"走吧。"他第二遍的声音比第一遍轻。
远处又传来了吼叫声。方向不对。不是他们来时的路。
```
